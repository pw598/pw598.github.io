Partial Derivatives - Linear Approximation and Linearizing

Linear approximation equations can help approximate a function's value when finding the exact value would be difficult. The linear approximation equation is like a tangent line equation. It intersects the function at the point of tangency, and helps us estimate the value of the function near the point of tangency.

Linear Approximation in Two Variables

We can use the linear approximation formula:

$L(x,y) = f(a,b) + \frac{\partial f}{\partial x}(a,b)(x-a) + \frac{\partial f}{\partial y}(a,b)(y-b)$

  • $(a,b)$ is the given point
  • $f(a,b)$ is the value of the function at $(a,b)$
  • $\frac{\partial f}{\partial x}(a,b)$ is the partial derivative of f with respect to $x$ at $(a,b)$
  • $\frac{\partial f}{\partial y}(a,b)$ is the partial derivative of f with respect to $y$ at $(a,b)$

Example:

Find the linear approximation of the multivariable function:

$f(x,y) = 6x^3 - 2xy^2, \text{ at } (1,2)$


$f(1,2) = 6(1)^3 - 2(1)(2)^2$

$f(1,2) = 6 - 2(4)$

$f(1,2) = -2$

Then we need to find the partial derivatives of the function with respect to $x$ and $y$.

$\frac{\partial f}{\partial x} = 18x^2 - 2y^2$

$\frac{\partial f}{\partial x}(1,2) = 18(1)^2 - 2(2)^2$

$\frac{\partial f}{\partial x}(1,2) = 10$

and

$\frac{\partial f}{\partial y} = -4xy$

$\frac{\partial f}{\partial y}(1,2) = -4(1)(2)$

$\frac{\partial f}{\partial y}(1,2) = -8$

Plugging the slope in each direction, $(a,b)$, and $f(a,b)$ into the linear approximation formula, we get:

$L(x,y) = -2 + (10)(x-1) + (-8)(y-2)$

$L(x,y) = -2 + 10x - 10 - 8y + 16$

$L(x,y) = 10x - 8y + 4$

This is the linear approximation of the given function at the given point.

Linearization of a Multivariable Function

Example:

Find the linearization of the following multivariable function at the following point.

$f(x,y) = 1 + x ~ln(xy - 5) \text{ at } (2,3)$

The equation for the linearization of a function is the same as what we used for the linear approximation in two variables.

$L(x,y) = f(a,b) + \frac{\partial f}{\partial x}(a,b)(x-a) + \frac{\partial f}{\partial y}(a,b)(y-b)$

$\frac{\partial f}{\partial x} = (1) ~ln(xy - 5) + (x) \left[ \frac{1}{xy-5} y \right]$

$\frac{\partial f}{\partial x}(2,3) = ln(2(3)-5) + \frac{(2)(3)}{(2)(3) - 5}$

$\frac{\partial f}{\partial x}(2,3) = ln(6-5) + \frac{6}{6 - 5}$

$\frac{\partial f}{\partial x}(2,3) = \frac{6}{1} = 6$

$\frac{\partial f}{\partial y} = x \frac{1}{xy-5} x$

$\frac{\partial f}{\partial y} = \frac{x^2}{xy-5}$

$\frac{\partial f}{\partial y}(2,3) = \frac{4}{6-5} = \frac{4}{1} = 4$

Our last step before we can plug everything we found into our linearization equation is to find $(a,b)$ by plugging the point we are given to our original function.

$f(2,3) = 1 + 2 ~ln(6-5)$

$f(2,3) = 1 + 2 ~ln(1)$

$f(2,3) = 1$


$L(x,y) = 1 + 6(x-2) + 4(y-3)$

$L(x,y) = 1 + 6x - 12 + 4y - 12$

$L(x,y) = 6x + 4y - 23$

Partial Derivatives - Gradient Vectors

Gradient Vectors

To find the gradient (a.k.a. the gradient vector) of a two variable function, we'll use the formula:

$\nabla f = \biggl\langle \frac{\partial f}{\partial x}, \frac{\partial f}{\partial y} \biggr\rangle$

This gives a vector-valued function that describes the function's gradient everywhere. To find the gradient at a particular point, we evaluate at that point.

$\nabla f(x,y) = \biggl\langle \frac{\partial f}{\partial x}(x,y), \frac{\partial f}{\partial y}(x,y) \biggr\rangle$

The maximal directional derivative is given by the magnitude of the gradient.

$||\nabla f|| = ||a,b|| = \sqrt{a^2 + b^2}$

where $a$ and $b$ come from $\nabla f(x,y) = \langle a,b \rangle$

The gradient $\nabla f$ always points in the direction of the maximal directional derivative. We can modify the formula to accomodate more variables as needed.

Example:

Find the maximal directional derivative and the direction in which it occurs, for:

$f(x,y) = x^3 + 2x^2y + 4y^2, \text{ at } P(1,1)$

We'll start with the partial derivatives of the given function $f$.

$\frac{\partial f}{\partial x} = 3x^2 + 4xy$

$\frac{\partial f}{\partial y} = 2x^2 + 8y$

The gradient of the function in general is:

$\nabla f = \biggl\langle \frac{\partial f}{\partial x}, \frac{\partial f}{\partial y} \biggr\rangle$

$\nabla f = \langle 3x^2 + 4xy, 2x^2 + 8y \rangle$

To find the gradient at the point we are interested in, we'll plug in $P(1,1)$.

$\nabla f(1,1) = \langle 3(1)^2 + 4(1)(1), 2(1)^2 + 8(1) \rangle$

$\nabla f(1,1) = \langle 7,10 \rangle$

To find the maximal directional derivative, we take the magnitude of the gradient that we found.

$|| \nabla f || = ||a,b|| = \sqrt{a^2 + b^2}$

$||7,10|| = \sqrt{(7)^2 + (10)^2}$

$||7,10|| = \sqrt{149}$

The maximal directional derivative always point in the direction of the gradient, so the maximal directional derivative is $||7,10|| = \sqrt{149}$, and it points toward $\nabla f(1,1) = \langle 7,10 \rangle$.

Gradient Vectors and the Tangent Plane

We learned that to find the gradient vector at a specific point, we use the formula:

$\nabla f(x,y,z) = \biggl\langle \frac{\partial f}{\partial x}, \frac{\partial f}{\partial y}, \frac{\partial f}{\partial z} \biggr\rangle$

where $(x,y,z)$ is the point we are interested in. If the result of evaluating the gradient vector at the point $(x,y,z)$ gives us $\nabla f(x,y,z) = \langle a,b,c \rangle$, then $a$, $b$, and $c$ represent the slope of the orginal function in the $x$, $y$, and $z$ directions.

Therefore, if interested in finding the equation of the tangent plane at $P(x_0, y_0, z_0)$, then we plug the values of $a$, $b$, and $c$, along with the point $P(x_0, y_0, z_0)$ into the formula for the equation of the tangent plane.

$a(x-x_0) + b(y-y_0) + c(z-z_0) = 0$

where $a, b, c$ come from $\nabla f(x,y,z) = \langle a,b,c \rangle$ and $x_0, y_0, z_0$ come from the given point $P(x_0, y_0, z_0)$.

Example:

Find the gradient vector of the function and use it to find the equation of the tangent plane at $P(3,4,-287)$.

$s^4 - 5x^3y - y^2 + 3y^4 + z = 6$

First, rearrange the function.

$f(x,y,z) = x^4 - 5x^3y - y^2 + 3y^4 + z - 6$

Now we'll start with the partial derivatives of the given function $f$.

$\frac{\partial f}{\partial x} = 4x^3 - 15x^2y$

$\frac{\partial f}{\partial y} = -5x^3 - 2y + 12y^3$

$\frac{\partial f}{\partial z} = 1$

To find the gradient vector at the point we are interested in, we'll plug the partial derivatives into the formula for the gradient vector, and then evaluate at the point of interest.

$\nabla f(x,y,z) = \biggl\langle \frac{\partial f}{\partial x}, \frac{\partial f}{\partial y}, \frac{\partial f}{\partial z} \biggr\rangle$

$\nabla f(x,y,z) = \langle 4x^3 - 15x^2y, -5x^3 - 2y + 12y^3, 1 \rangle$

Evaluating at $P(3,4,-287)$, we get:

$\nabla f(3,4,-287) = \langle 4(3)^3 - 15(3)^2(4), -5(3)^3 - 2(4) + 12(4)^3, 1 \rangle$

$\nabla f(3,4,-287) = \langle -432, 625, 1 \rangle$

Now we have everything needed to find the equation of the tangent plane.

$a(x-x_0) + b(y-y_0) + c(z-z_0) = 0$

$-432(x-3) + 625(y-4) + 1(z + 287) = 0$

$-432x + 1296 + 625y - 2500 + z + 287 = 0$

$-432x + 625y + z = 917$

Maximum Rate of Change and Its Direction

Example:

Find the maximum rate of change of the following function at the given point.

$f(x,y) = sin(xy) \text{ at } (1,0)$

The first thing we need to do is find the gradient vector. Then, we'll use it to find the maximum rate of change. The maximum rate of change will always occur in the gradient direction of the gradient vector.

First, we'll take partial derivatives. We'll use the chain rule, and treat $sin$ as our outside function and $xy$ as our inside function.

$\frac{\partial f}{\partial x} = cos(xy) \cdot y = y ~cos(xy)$

$\frac{\partial f}{\partial y} = cos(xy) \cdot x = x ~cos(xy)$

Our gradient vector is the partial derivative with respect to $x$ and with respect to $y$.

$\nabla f(x,y) = \langle y ~cos(xy), x ~cos(xy) \rangle$

$\nabla f(1,0) = \langle 0,1 \rangle$

$|\nabla f(1,0)| = \sqrt{0^2 + 1^2}$

The max rate of change is equal to 1, and that maximum rate of change occurs in the direction of the gradient vector, which is the vector $\langle 0,1 \rangle$.

Partial Derivatives - Tangent Planes and Normal Lines

The tangent plane is the 3D version of the tangent line. It is a flat surface that intersects the graph at only one point. To find the equation of the tangent plane to a 3D function at a specific point, use the formula:

$z - z_1 = \frac{\partial z}{\partial x}(x_1, y_1, z_1)(x-x_1) + \frac{\partial z}{\partial y}(x_1, y_1, z_1)(y-y_1)$

where

$\frac{\partial z}{\partial x}(x_1, y_1, z_1)$ is the partial derivative of $z$ with respect to $y$ at the point $(x_1, y_1, z_1)$.

There are 3 steps to managing tangent plane problems:

  1. Find the partial derivatives of the function with respect to each variable
  2. Evaluate the partial derivatives at the given point to find the slope in each direction
  3. Plug the slopes and the given point into the formula for the equation of the tangent plane

Example:

Find the equation of the tangent plane of the following function and given point.

$z = 3 + \frac{x^2}{16} + \frac{y^2}{9}, \text{ at } P(-4,3,5)$

We'll start by finding partial derivatives of the function $z$ with respect to $x$ and $y$.

$\frac{\partial z}{\partial x} = \frac{2x}{16}$

$\frac{\partial z}{\partial x} = \frac{x}{8}$

and

$\frac{\partial z}{\partial y} = \frac{2y}{9}$

Plugging the given point $P(-4,3,5)$ into the partial derivatives, we'll get values for the slope in our direction.

$\frac{\partial z}{\partial x} ~P(-4,3,5) = \frac{-4}{8} = \frac{-1}{2}$

$\frac{\partial z}{\partial y} ~P(-4,3,5) = \frac{2(3)}{9} = \frac{2}{3}$

Now we'll plug the slope in each direction and the point $P(-4,3,5)$ into the formula for the equation of the tangent plane.

$z - z_1 = \frac{\partial z}{\partial x} (x_1, y_1, z_1)(x-x_1) + \frac{\partial z}{\partial y}(x_1, y_1, z_1)(y-y_1)$

$z - 5 = - \frac{1}{2}[x - (-4)] + \frac{2}{3}(y-3)$

$z - 5 = - \frac{1}{2}(x+4) + \frac{2}{3}y - 2$

$z - 5 = - \frac{1}{2}x - 2 + \frac{2}{3}y - 2$

$z = - \frac{1}{2}x = \frac{2}{3}y + 1$

Normal Line to the Surface

The normal line to the surface of the tangent plane is the line which is set at $90^{\circ}$ from the surface. In order to find the equation of the normal line to a tangent plane, we aren't required to first find the equation of the tangent plane, but both the normal line and tangent plane use the derivatives of the original function and the point of tangency.

The tangent plane is given by:

$\frac{\partial f}{\partial x}(x_0, y_0, z_0)(x-x_0) + \frac{\partial f}{\partial y}(x_0, y_0, z_0)(y-y_0) + \frac{\partial f}{\partial z}(x_0, y_0, z_0)(z-z_0) = 0$

where $(x_0, y_0, z_0)$ is the point of tangency.

The normal line to the surface is given by:

$\frac{x - x_0}{\partial f / \partial x} = \frac{y - y_0}{\partial f / \partial y} = \frac{z - z_0}{\partial f / \partial z}$

Example:

For the function $x^2 + y^2 + 2z^2 - 36$

  • Find the equation of the tangent plane at the point $P(-1,2,4)$
  • Find the normal line to the tangent plane at the point $P(-1,2,4)$

We must find the partial derivatives of the function $f$ with respect to each variable at the point $P(-1,2,4)$.

$\frac{\partial f}{\partial x} = 2x$

$\frac{\partial f}{\partial x}(-1,2,4) = 2(-1)$

$\frac{\partial f}{\partial x}(-1,2,4) = -2$

and

$\frac{\partial f}{\partial y} = 2y$

$\frac{\partial f}{\partial y}(-1,2,4) = 2(2) = 4$

$\frac{\partial f}{\partial z} = 4z$

$\frac{\partial f}{\partial z}(-1,2,4) = 4(4) = 16$

Plugging the slope in each direction and the point $P(-1,2,4)$ into the equation of the tangent plane gives:

$-2[x - (-1)] + 4(y-2) + 16(z-4) = 0$

$-2(x+1) + 4(y - 2) + 16(z-4) = 0$

$-2x - 2 + 4y - 8 + 16z - 64 = 0$

$-2x + 4y + 16z = 74$

$-x + 2y + 8z = 37$

To find the normal line to the tangent plane $-x + 2y + 8z = 37$ at the point $P(-1,2,4)$ into the symmetric formula:

$\frac{x-x_0}{\partial f / \partial x} = \frac{y-y_0}{\partial f / \partial y} = \frac{z-z_0}{\partial f / \partial z}$

$\frac{x - (-1)}{-2} = \frac{y-2}{4} = \frac{z-4}{16}$

$\frac{x+1}{-2} = \frac{y-2}{4} = \frac{z-4}{16}$

This is the normal line to the plane for the tangent plane $-x + 2y + 8z = 37$.

Partial Derivatives - Optimization

Optimization is the process of finding the absolute highest and lowest points of a function. With multivariable functions, it is about finding the highest and lowest points along the 3D surface.

Second Derivative Test

As with single variable functions, we can use the second derivative test with multivariable functions to classify any critical points that the function has.

To use the second derivative test, we'll take the partial derivatives of the function with respect to each variable, then set them equal to $0$ and use them as a system of equations to solve for the coordinates of all possible points.

$D(x,y) = \frac{\partial^2 f}{\partial x^2} \cdot \frac{\partial^2 f}{\partial y^2} - \left( \frac{\partial^2 f}{\partial x \partial y} \right)^2$

If $D(x,y) \lt 0$, then $f$ has a saddle point at $(x,y)$.

If $D(x,y) = 0$, then the test is inconclusive

If $D(x,y) \gt 0$, then:

  • If $\frac{\partial^2 f}{\partial x^2} \gt 0$ then $f$ has a local minimum at $(x,y)$
  • If $\frac{\partial^2 f}{\partial x^2} \lt 0$, then $f$ has a local maximum at $(x,y)$

Example:

Find and classify the critical points of the following function:

$f(x,y) = 6 + x^3 + y^3 - 3xy$

We'll start by finding the first-order partial derivatives.

$\frac{\partial f}{\partial x} = 3x^2 - 3y$

$\frac{\partial f}{\partial y} = 3y^2 - 3x$

Setting both to $0$ and using them as a system of simultaneous equations to find the critical points gives:

$3x^2 - 3y = 0$$3y - 3x = 0$
$3x^2 = 3y$ 
$y = x^2$ 
 $3(x^2)^2 - 3x = 0$
 $3x^4 - 3x = 0$
 $3x(x^3 - 1) = 0$
 $x = 0 or x = 1$
$y = (0)^2, y = (1)^2$ 
$y = 0, y = 1$ 
$(0,0), (1,1)$

Plugging these into the second derivative test formula, we get:

$D(x,y) = \frac{\partial^2 f}{\partial x} \cdot \frac{\partial^2 f}{\partial y} - \left( \frac{\partial^2 f}{\partial x \partial y} \right)^2$

$D(x,y) = (6x)(6y) - (-3)^2$

$D(x,y) = 36xy - 9$

Now we can use the equation for $D(x,y)$ to test our critical points one at a time.

For $(0,0)$;

  • $D(0,0) = 36(0)(0) - 9 $
  • $D(0,0) = -9$

Since $-9 \lt 0$, the function has a saddle point at $(0,0)$.

For $(1,1)$:

  • $D(1,1) = 36(1)(1) - 9$
  • $D(1,1) = 27$

Since $27 \gt 0$, we must look at $\partial^2 f / \partial x^2$ to evaluate $(1,1)$ further.

$\frac{\partial^2 f}{\partial x^2} = 6x$

$\frac{\partial^2 f}{\partial x^2}(1,1) = 6$

Since $6 \gt 0$, the function has a local minimum at $(1,1)$.

To summarize, there is a saddle point at $(0,0)$, and a local minimum at $(1,1)$.

Local Extrema and Saddle Points

The way we'll find local maxima and minima is using the second derivative test. In order to use that test, we'll need to find all the second order partial derivatives of $f(x,y)$, which requires first finding all the first-order partial derivatives.

Example:

Find the local maxima and minima of the following function.

$f(x,y) = x^3 - 12xy + 8y^3$


$\frac{\partial f}{\partial x} = 3x^2 - 12y$

$\frac{\partial^2 f}{\partial x^2} = 6x$


$\frac{\partial f}{\partial y} = -12x + 24y^2$

$\frac{\partial^2 f}{\partial y^2} = 48y$


$\frac{\partial^2 f}{\partial x \partial y} = -12$

Now that we have the first and second order partial derivatives, the next step is to find critical points of this 3D function. We start by setting the first order partial derivatives to $0$.

$3x^2 - 12y = 0$

$x^2 - 4y = 0$

and

$-12x + 24y^2 = 0$

$x + 2y^2 = 0$

Now we want to use these equations as a system of simultaneous equations. We can start by solving $12x - 24y^2 = 0$ for $x$.

$12x - 24y^2 = 0$

$2y^2 = x$

Then plug that value of $x$ into the other equation, and solve for $y$.

$x^2 - 4y = 0$

$(2y^2)^2 - 4y = 0$

$4y(y^3 - 1) = 0$

$y = 0 or y = 1$

Now we need to find corresponding values of $x$ for both of these $y$ values, plugging them into $2y^2 = x$.

$x = 2(0)^2$

$x = 0$

and

$x = 2(1)^2$

$x = 2$

In order to use the second derivative test to evaluate these critical points, to see whether or not either is a local minimum or maximum, we need to plug these points into the second order partial derivatives.

$\frac{\partial^2 f}{\partial x^2} = 6x$

$\frac{\partial^2 f}{\partial x^2}(0,0) = 0$

$\frac{\partial^2 f}{\partial x^2}(2,1) = 12$


$\frac{\partial^2 f}{\partial y^2} = 48y$

$\frac{\partial^2 f}{\partial y^2}(0,0) = 0$

$\frac{\partial^2 f}{\partial y^2}(2,1) = 48$


$\frac{\partial^2 f}{\partial x \partial y} = -12$

$\frac{\partial^2 f}{\partial x \partial y}(0,0) = -12$

$\frac{\partial^2 f}{\partial x \partial y}(2,1) = -12$


$D(0,0) = 0 \cdot 0 - (-12)^2$

$D(0,0) = -144$

Because $-144 \lt 0$, point $(0,0)$ is a saddle point; neither a minima nor maxima.

$D(2,1) = 12 \cdot 48 - (-12)^2$

$D(2,1) = 432$

Because $432 \gt 0$, we look at the partial derivative of x at point $(2,1)$. Because its value of $12$ is greater than $0$, we conclude that the point $(2,1)$ is a local minimum.

Global Extrema

Here we will use the second derivative test to find the global extrema of a function.

Example:

Find the global extrema of the following function.

$f(x,y) = 2x^2 + 3y^2 - 12x - 6y + 9$

Start by finding the partial derivatives.

$\frac{\partial f}{\partial x} = 4x - 12$

$\frac{\partial f}{\partial y} = 6y - 6$

We set both of these equal to $0$ and solve for our variables.

$4x - 12 = 0$

$4x = 12$

$x = 3$


$6y - 6 = 0$

$6y = 6$

$y = 1$

This generates the critical point $(3,1)$. We want to use the second derivative test to see whether this point represents a local minimum or maximum. Because it is the only critical point, this is the only point upon which the function changes direction, and will be the global minimum or maximum. If we had multiple critical points, we would need to test each one and then plug them back into the original function to see which critical point represents the higher or lower value.

We'll need to plug $(3,1)$ into the following formula for $D$.

$D = (D_{xx})(D_{yy}) - (D_{xy})^2$

This means $D$ is equal to the second order partial derivative of $f$ with respect to $x$, multiplied by the second order derivative of $f$ with respect to $y$, minus the mixed second-order partial derivative.

$\frac{\partial^2 f}{\partial x^2} = 4$

$\frac{\partial^2 f}{\partial y^2} = 6$

$\frac{\partial^2 f}{\partial x \partial y} = 0$


$D = (4)(6) - (0)^2$

$D = 24$

If there were multiple critical points, the second order partial derivatives would still include variables, in which case we would need to plug the critical point into the second order partial derivatives, before plugging them into the formula for $D$. In this case we have a value of $D=24$ for the critical point $(3,1)$.

Since $D \gt 0$, we have to look at the second order partial derivative for $x$. It equals $4$, which is greater than $0$, so the critical point $(3,1)$ represents a local minimum. Because it is the only critical point, it is also the global minimum.

Extreme Value Theorem

In this problem, we will discuss how to find global extremes of a multivariable function, given that this function has been defined for some domain $D$.

Example:

Find the global extrema of the following function in the following domain.

$f(x,y) = x^2 + y^2 + x^2y + 4$

$D = {|x| \le 1, |y| \le 1}$

There will be 3 steps in examining the domain space for critical points: 1) examining the middle, 2) examining the edges, and 3) examining the corners.

$\frac{\partial f}{\partial x} = 2x + 2y$

$\frac{\partial f}{\partial y} = 2y + x^2$

In order to find critical points, we'll need to solve one of these equations for one of our variables. We'll start arbitrarily with the second equation.

$\frac{\partial f}{\partial y} = 2y + x^2$

$2y = -x^2$

$y = - \frac{1}{2}x^2$


$\frac{\partial f}{\partial x} = 2x + 2xy$

$2x + 2x \left( - \frac{1}{2} x^2 \right) = 0$

$2x - x^3 = 0$

$x(2-x^2) = 0$

$x = 0 \text{ or } x = \pm \sqrt{2}$

However, $\pm \sqrt{2}$ is outside of the domain, so we only consider $x=0$.

$y = - \frac{1}{2}x^2$

$y = - \frac{1}{2}(0)^2$

The only critical point is $(0,0)$. We'll plug this into the original function.

$f(0,0) = (0)^2 + (0)^2 + (0)^2(0) + 4$

$f(0,0) = 4$

Next, we'll test the corners of the domain.

$f(1,1) = (1)^2 + (1)^2 + (1)^2(1) + 4$

$f(1,1) = 7$


$f(1,-1) = (1)^2 + (-1)^2 + (1)^2(-1) + 4$

$f(1,-1) = 5$


$f(-1,-1) = (-1)^2 + (-1)^2 + (-1)^2(-1)$

$f(-1,-1) = 5$


$f(-1,1) = (-1)^2 + (1)^2 + (-1)^2(1) + 4$

$f(-1,1) = 7$

We'll label the edges as follows:

$D_1$: corresponds to the line between $(-1,-1)$ and $(1,-1)$

$D_2$: corresponds to the line between $(1,-1)$ and $(1,1)$

$D_3$: corresponds to the line between $(-1,1)$ and $(1,1)$

$D_4$: corresponds to the line between $(-1,-1)$ and $(-1,1)$


$D_1: f(x,-1) = x^2 + 1 - x^2 + 4 = 5$

$D_2: f(1,y) = 1 + y^2 + y + 4$

$D_2: f(1,y) = y^2 + y + 5$

Because the $y$ variable remains, we take the partial derivative with respect to $y$, and set it to zero.

$\frac{\partial f}{\partial y}(1,y) = 2y + 1 = 0$

$2y = -1$

$y = - \frac{1}{2}$

This provides the critical point $\left( 1, - \frac{1}{2} \right)$

$D_3: f(x,1) = x^2 + 1 + x^2 + 4$

$D_3: f(x,1) = 2x^2 + 5$


$\frac{\partial f}{\partial x} = 4x = 0$

$x = 0$

This provides the critical point $(0,1)$.

$D_4: f(-1,y) = 1 + y^2 + y + 4 = 0$

$D_4: f(-1,y) = y^2 + y = 5$

$\frac{\partial f}{\partial y} = 2y + 1 = 0$

$2y = -1$

$y = - \frac{1}{2}$

This provides the critical point $\left( 1, - \frac{1}{2} \right)$

Now to test the critical points:

$f(0,1) = 1 + 4 = 5$

$f \left( -1, - \frac{1}{2} \right) = 1 + \frac{1}{4} - \frac{1}{2} + 4 = 4 \frac{3}{4}$

$f \left( 1, - \frac{1}{2} \right) = 1 + \frac{1}{4} - \frac{1}{2} + 4 = 4 \frac{3}{4}$

$f(0,0) = 4$

$f \left( \pm 1, - \frac{1}{2} \right) = 4 \frac{3}{4}$

$f(0,1), f(\pm 1, 1) = 5$

$f(\pm 1, 1) = 7$

There is a global minimum at point $(0,0)$, equal to $4$.

There are maxima at points $(\pm 1, 1)$, equal to $7$.

Partial Derivatives - Applied Optimization

Maximum Product of 3 Real Numbers

Example:

Find 3 positive numbers with sum $100$, and the maximum product of these 3 numbers.

$x + y + z = 100$

$p = xyz$

To maximize the equation for the product, we need to reduce it to at most two variables, by solving the top equation for one of these variables, and then plugging it into the second equation.

$z = 100 - x - y$

$p = xy(100 - x - y)$

$f(x,y) = 100xy - x^2y - xy^2$

$\frac{\partial f}{\partial x} = 100y - 2xy - y^2$

$\frac{\partial f}{\partial y} = 100x - x^2 - 2xy$

$\frac{\partial^2 f}{\partial x^2} = -2y$

$\frac{\partial^2 f}{\partial y^2} = -2x$

$\frac{\partial^2 f}{\partial x \partial y} = 100 - 2x - 2y$

Now, what we're looking to do is use our first order partial derivatives to find critical points of $f(x,y)$. We'll do that by solving this as a system of equations.

$\frac{\partial f}{\partial x} = 100y - 2xy - y^2$

$\rightarrow y(100 - 2x - y) = 0$

$\frac{\partial f}{\partial y} = 100x - x^2 - 2xy$

$\rightarrow x(100 - x - 2y) = 0$

We'll find values of $y$ that satisfy the first equation above, then plug these into the second equation to find corresponding values of $x$.

$y(100 - 2x - y) = 0$

$y = 0 \text{ or } y = 100 - 2x$


For $x = 0$:

$x(100 - x - 2y) = 0$

$x(100 - x - 2(0)) = 0$

$x(100 - x) = 0$

$x = 0 \text{ or } x = 100$


For $x = 100$:

$x(100 - x - 2y) = 0$

$x(100 - x - 2)(100 - 2x)) = 0$

$x(100 - x - 200 + 4x) = 0$

$x(-100 + 3x) = 0$

$3x = 100$

$x = \frac{100}{3}$


$y = 100 - 2x$

$y = 100 - \frac{200}{3}$

$y = \frac{100}{3}$

We have generated four critical points:

$\left( \frac{100}{3}, \frac{100}{3} \right)$

$(0,100)$

$(0,0)$

$(100,0)$

Three of them contain zeros, and therefore cannot be a solution to the original question, leaving critical point $\left( \frac{100}{3}, \frac{100}{3} \right)$. We will use the second derivative test to ensure that this is indeed a maximum.

$D = (D_xx)(D_yy) - D(xy)^2$

$ D = \left( -2 \left( \frac{100}{3} \right) \right) \left( -2 \left( \frac{100}{3} \right) \right) - \left( 100 - 2 \left(\frac{100}{3} \right) - 2 \left( \frac{100}{3} \right) \right) $

$D = \left( - \frac{}{} \right) \left( - \frac{}{} \right) - \left( 100 - \frac{200}{3} - \frac{200}{3} \right)$

$D = \frac{40000}{9} - \left( \frac{10000}{9} \right)$

$D = \frac{30000}{9}$

The answer is greater than zero, so we have to look at the partial derivative with respect to x, evaluated at the critical point.

$\frac{\partial^2 f}{\partial^2 x} \left( \frac{100}{3} \right) \left( \frac{100}{3} \right) = - \frac{200}{3}$

$\frac{-200}{3} \lt 0$, so there is a maximum at the critical point found.

Now to find $z$, using the equation $z = 100 - x - y$.

$z = 100 - \frac{100}{3} - \frac{100}{3}$

$z = \frac{100}{3}$

The 3 numbers with sum $100$ and maximum product are $\left( \frac{100}{3}, \frac{100}{3}, \frac{100}{3} \right)$.

Minimum Distance from a Point to a Plane

Example:

Find the smallest distance between a point $(2,1,1)$ and a plane $x + 2y + 2z = 11$.

Recall that when $ax + by + cz + d = 0$, given a point $P(x_0, y_0, z_0)$, then:

$D = \frac{ | ax_0 + by_0 + cz_0 + d | }{ \sqrt{ a^2 + b^2 + c^2 } }$

We'll rewrite $x + 2y + 2z = 11$ so that the constant is on the left side.

$x + 2y + 2z - 11 = 0$

We then plug this and the given point into the distance formula.

$D = \frac{ | (1)(2) + (2)(1) + (2)(1) + (-11) | }{ \sqrt{ 1^2 + 2^2 + 2^2 } }$

$D = \frac{ | 2 + 2 + 2 + 11 | }{ \sqrt{ 1 + 4 + 4 } }$

$D = \frac{|-5|}{\sqrt{9}}$

$D = \frac{5}{\sqrt{9}}$

$D = \frac{5}{3}$

Points on the Cone Closest to a Given Point

Example:

Find the point on the cone $z^2 = x^2 + y^2$ closest to the point $(4,2,0)$.

Recall the distance formula:

$D = \sqrt{ (x - x_1)^2 + (y - y_1)^2 + (z - z_1)^2 }$

We want to eliminate one of the variables by expressing it in terms of the others (knowing that $z^2 = x^2 + y^2$).

$D = \sqrt{ (x - 4)^2 + (y - 2)^2 + z^2 }$

$D = \sqrt{ (x - 4)^2 + (y - 2)^2 + x^2 + y^2 }$

$D^2 = (x-4)^2 + (y - 2)^2 + x^2 + y^2$

$D^2 = (x^2 - 8x + 16) + y^2 - 4y + 4 + x^2 + y^2$

$D^2 = 2x^2 - 8x + 2y^2 - 4y + 20$

Now we take the partial derivatives, and set their equations to $0$.

$\frac{\partial D^2}{\partial x} = 4x - 8 = 0$

$4x = 8$

$x = 2$


$\frac{\partial D^2}{\partial y} = 4y - 4 = 0$

$4y = 4$

$y = 1$

This tells us there is a critical point $(2,1)$. We then plug it into the formula for $z^2$.

$z^2 = x^2 + y^2$

$z^2 = 4 + 1$

$z^2 = 5$

$z = \pm \sqrt{5}$

This tells us two points on the cone are equidistant; they are $(2,1,5)$ or $(2,1,-5)$.

If we want to use the second derivative test to verify that this critical point represents the shortest distance between the given point and the cone, find the partial derivatives and plug them into the formula for the second derivative test.

$D^2_{xx} = 4$

$D^2_{yy} = 4$

$D^2_{xy} = 0$

$SDT = 4 \cdot 4 - (0)^2$

$SDT = 16$

Because the second derivative test gives a result greater than $0$, we have to look at the second order partial derivative with respect to $x$. It is greater than $0$, so the point is a minimum, proving shortest distance to the cone.

Partial Derivatives - Lagrange Multipliers

Lagrange Multipliers

Sometimes we are asked to find and classify the critical points of a multivariable function that is subject to a secondary constraint equation.

  • Multivariable Functions: $z = f(x,y)$
  • Constraint Equation = $g(x,y) = c$

To find the critical points of a function like this one, we can use the Lagrange multiplier $\lambda$ to develop a system of simultaneous equations that will allow us to solve for critical points.

  1. Bring the constant $c$ in $g(x,y)$ to the left-hand side, so that you end up with two functions in the forms of $f(x,y)$ and $g(x,y)$.
  2. Take the partial derivatives of $f(x,y)$ and $g(x,y)$.
  3. Multiply the partial derivatives of $g(x,y)$ by $\lambda$.
  4. Create the system of simultaneous equations $\frac{\partial f}{\partial x} = \frac{\partial g}{\partial x} \lambda$ and $\frac{\partial f}{\partial y} = \frac{\partial g}{\partial y} \lambda$.
  5. Solve both equations for $\lambda$, then set them equal to each other.
  6. Solve for one variable in terms of the other, then plug the original constraint equation in to find values for both $x$ and $y$.
  7. Use the second derivative test to classify the critical point

$D(x,y,\lambda) = \frac{\partial^2 f}{\partial x^2} \cdot \frac{\partial^2 f}{\partial y^2} - \left( \frac{\partial^2 f}{\partial x \partial y} \right)^2$

If $D(x,y) \lt 0$, then $f$ has a saddle point at $(x,y)$.

If $D(x,y) = 0$, then the test is inconclusive

If $D(x,y) \gt 0$, then:

  • If $\frac{\partial^2 f}{\partial x^2} \gt 0$, $f$ has a local minimum at $(x,y)$
  • If $D(x,y) \gt 0$, and $\frac{\partial^2 f}{\partial x^2} \lt 0$, then $f$ has a local maximum at $(x,y)$

Example:

Find the maximum and minimum of the following function with the given constraint.

$f(x,y) = x^2 + 2y^2 - 120, \text{ with constraint } 2y - 4x = 24$

We need to set the constraint equation equal to $0$ to get it into the same form as our function.

$g(x,y) = 2y - 4x - 24$

Now we can find the partial derivatives of the function $f(x,y)$ and the constraint equation $g(x,y)$.

For $f(x,y)$:

$\frac{\partial f}{\partial x} = 2x \text{ and } \frac{\partial f}{\partial y} = 4y$

For $g(x,y)$:

$\frac{\partial g}{\partial x} = -4 \text{ and } \frac{\partial g}{\partial y} = 2$

We'll multiply the partial derivatives of the constraint equation by $\lambda$, and get:

$\frac{\partial g}{\partial x} \lambda = -4 \lambda \text{ and } \frac{\partial g}{\partial y} \lambda = 2 \lambda$

Next, create the system of simultaneous equations.

$\frac{\partial f}{\partial x} = \frac{\partial g}{\partial x} \lambda$

$2x = -4 \lambda$

$\lambda = - \frac{x}{2}$

and

$\frac{}{} = \frac{}{} \lambda$

$4y = 2 \lambda$

$\lambda = 2y$

With two equations for $\lambda$, we can set them equal to one another, and solve for $y$ in terms of $x$.

$2y = - \frac{x}{2}$

$y = - \frac{x}{4}$

Plugging this y-value into the constraint equation and solving for $x$, we get:

$2y - 4x = 24$

$2 \left( - \frac{x}{4} \right) - 4x = 24$

$- \frac{x}{2} - 4x = 24$

$-x - 8x = 48$

$-9x = 48$

$x = - \frac{48}{9}$

Now we can plug this real-number value of $x$ into the equation to find a real-number value of $y$.

$2y - 4 \left( - \frac{48}{9} \right) = 24$

$2y + \frac{192}{9} = 24$

$2y + \frac{64}{3} = 24$

$6y + 64 = 72$

$6y = 8$

$y = \frac{4}{3}$

This tells us that our single critical point is:

$\left( - \frac{48}{9}, \frac{4}{3} \right)$

Now, to classify this critical point, we'll find the second-order partial derivatives of $f$.

$\frac{\partial^2 f}{\partial x^2} = 2$

$\frac{\partial^2 f}{\partial y^2} = 4$

$\frac{\partial^2 f}{\partial x \partial y} = 0$

Plug the second-order partial derivatives into the second derivative test formula.

$D(x,y,\lambda) = \frac{\partial^2 f}{\partial x^2} \cdot \frac{\partial^2 f}{\partial y^2} - \left( \frac{\partial^2 f}{\partial x \partial y} \right)^2$

$D(x,y,\lambda) = (2)(4)-(0)^2$

$D(x,y,\lambda) = 8$

Since we have no variables remaining in $D(x,y,\lambda)$, we'll have the same result for all possible points. Because $D \gt 0$, we have to look at $\partial^2 f / \partial x^2$.

$\frac{\partial^2 f}{\partial x^2} \left( - \frac{48}{9}, \frac{4}{3} \right) = 2$

Since $2 \gt 0$, the function has a local minimum at this critical point.

Two Dimensions, One Constraint

Find any extrema of $f(x,y) = x^2 + y^2$ subject to $2x + 6y = 2000$.

Recall that:

$D(x,y,\lambda) = \left( \frac{\partial^2 f}{\partial x^2} \right) \left( \frac{\partial^2 f}{\partial y^2} \right) - \left( \frac{\partial^2 f}{\partial x \partial y} \right)^2$

The first thing we want to do is rewrite the constraint equation to make it an equation for $g(x,y)$.

$2x + 6y - 2000 = 0$

$g(x,y) = 2x + 6y - 2000$

The next thing we're going to do is find the first order partial derivatives of both $f$ and $g$.

$\frac{\partial f}{\partial x} = 2x$

$\frac{\partial f}{\partial y} = 2y$

$\frac{\partial g}{\partial x} = 2$

$\frac{\partial g}{\partial y} = 6$

With these partial rules in place, we introduce $\lambda$.

$2x = 2 \lambda$

$2y = 6 \lambda$

Now we solve for $\lambda$ in one of these equations, and plug the result into the other equation.

$2x = 2 \lambda$

$x = \lambda$


$2y = 6 \lambda$

$2y = 6x$

$y = 3x$

With this solved for $y$ in terms of $x$, we can take the $y$ value and plug it into the constraint equation.

$2x + 6y = 2000$

$2x + 6(3x) = 2000$

$2x + 18x = 2000$

$20x = 2000$

$x = 100$

We can now plug $x=100$ into the equation for $y$.

$y = 3(100)$

$y = 300$

We have found the critical point $(100,300)$, and will plug that into the second derivative test.

$\frac{\partial^2 f}{\partial x} = 2$

$\frac{\partial^2 f}{\partial y} = 3$

$\frac{\partial^2 f}{\partial x \partial y} = 0$

$D(x,y,\lambda) = (2)(2) - (0)^2 = 4$

Because $4 \gt 0$, we look at the partial derivative with respect to $x$. Since $2 \gt 0$, the critical point represents a local minimum.
