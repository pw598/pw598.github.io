Vectors - Vector Functions and Space Curves

Domain of a Vector Function

To find the domain of a vector function, $r(t) = \langle a,b,c \rangle$ or $r(t) = a \mathbf{i} + b \mathbf{j} + c \mathbf{k}$, we'll need to find the domain of the individual components $a$, $b$, and $c$. The domain of the vector function is the values for which the domains of $a$, $b$, and $c$ overlap.

Example:

Find the domain of the following vector function:

$r(t) = \biggl\langle ln(t-1), t^2, \frac{1}{9-t^2} \biggr\rangle$

The domain of $ln(t-1)$ is:

$t - 1 \gt 0$

$t \gt 1$

The domain of $t^2$ is all real numbers.

The domain of $\frac{1}{9-t^2}$ is:

$9 - t^2 \neq 0$

$-t^2 \neq -9$

$t^2 \neq -9$

$t \neq \pm 3$

The first component restricts the domain of the entire vector function to $t \gt 1$. The second component doesn't restrict the domain at all. The third component says $t$ cannot include $\pm 3$. $t = -3$ is outside of the domain $t \gt 1$, so we can ignore it.

The domain of the vector function is $t \gt 1$ but $t \neq 3$.

Limit of a Vector Function

To find the limit of a vector function, $r(t) = a(t) \mathbf{i} + b(t) \mathbf{j} + c(t) \mathbf{k}$, we need to take the limit of each term separately.

$\lim \limits_{t \to x} [ a(t) \mathbf{i} + b(t) \mathbf{j} + c(t) \mathbf{k} ]$

$= \lim \limits_{t \to x} a(t) \mathbf{i} + \lim \limits_{t \to x} b(t) \mathbf{j} + \lim \limits_{t \to x} c(t) \mathbf{k}$

$= a(x) \mathbf{i} + b(x) \mathbf{j} + c(x) \mathbf{k}$

Example:

Find the limit of the vector function:

$\lim \limits_{t \to 0} \left[ (t^2 - 2) \mathbf{i} + ln(t + e) \mathbf{j} + \frac{4t}{sin ~t} \mathbf{k} \right]$

We'll take the limit of each term separately.

$\lim \limits_{t \to 0}(t^2 - 2) \mathbf{i} + \lim \limits_{t \to 0} ln(t + e) \mathbf{j} + \lim \limits_{t \to 0} \frac{4t}{sin ~t} \mathbf{k}$

Evaluating the first two terms as $t \to 0$, we get:

$(0^2 - 2) \mathbf{i} + ln(0 + e) \mathbf{j} + \lim \limits_{t \to 0} \frac{4t}{sin ~t} \mathbf{k}$

$-2 \mathbf{i} + ln(e) \mathbf{j} + \lim \limits_{t \to 0} \frac{4t}{sin ~t}$

$-2 \mathbf{i} + \mathbf{j} + \lim \limits_{t \to 0} \frac{4t}{sin ~t} \mathbf{k}$

Because the third term gives $0/0$ when $t \to 0$, we use L'Hopital's rule, replacing the numerator and denominator with their derivatives.

$-2 \mathbf{i} + \mathbf{j} + \lim \limits_{t \to 0} \frac{4}{cos ~t} \mathbf{k}$

Evaluating as $t \to 0$, we get:

$-2 \mathbf{i} + \mathbf{j} + \frac{4}{cos ~0} \mathbf{k}$

$-2 \mathbf{i} + \mathbf{j} + \frac{4}{1} \mathbf{k}$

$-2 \mathbf{i} + \mathbf{j} + 4 \mathbf{k}$

This is the limit of the vector function.

Vector and Parametric Equations of a Line Segment

Sometimes we need to find the equation of a line segment when we only have the endpoints of the line segment. The vector equation of the line segment is given by:

$r(t) = (1-t)r_0 + tr_1$

where $0 \le t \le 1$ and $r_0$ and $r_1$ are the vector equivalents of the endpoints.

The parametric equations of the line segment are given by:

$x = r(t)_1$

$y = r(t)_2$

$z = r(t)_3$

where $r(t)_1$, $r(t)_2$, and $r(t)_3$ come from the vector function.

$r(t) = r(t)_1 \mathbf{i} + r(t)_2 \mathbf{j} + r(t)_3 \mathbf{k}$

$r(t) = \langle r(t)_1, r(t)_2, r(t)_3 \rangle$

Example:

Find the vector and parametric equations of the following line segment defined by its endpoints.

$P(1, 2, -1)$

$Q(1, 0, 3)$

We'll convert its endpoints to their vector equivalents.

$r_0 = \langle 1, 2, -1 \rangle$

$r_1 = \langle 1, 0, 3 \rangle$

Plugging these into the vector formula for the equation of the line segment gives:

$r(t) = (1-t) \langle 1, ~2, ~-1 \rangle + t \langle 1, 0, 3 \rangle$

$r(t) = \langle 1-t, ~2-2t, ~-1+t \rangle + \langle t, 0, 3t \rangle$

$r(t) = \langle 1-t+t, ~2-2t+0, ~-1 + t + 3t \rangle$

$r(t) = \langle 1, ~2-2t, ~-1 + 4t \rangle$

We can also write the vector equation as:

$r(t) = \mathbf{i} + (2 - 2t) \mathbf{j} + (-1 + 4t) \mathbf{k}$

Now that we have the vector equation of the line segment, we just take its direction numbers, or the coefficients \mathbf{i}, \mathbf{j}, and \mathbf{k} to get the parametric equations of the line segment.

$x = 1$

$y = 2 - 2t$

$z = -1 + 4t$

To summarize:

Vector Equation: $r(t) \langle 1, ~2-2t, ~-1 + 4t \rangle$

Parametric Equations: $x=1$, $y=2-2t$, and $z=1+4t$

Vector Function for the Curve of Intersection of Two Surfaces

When two 3D surfaces intersect each other, the intersection is a curve. We can find the vector equation of that curve using these steps:

1. Set the curves equal to each other and solve for one of the remaining variables in terms of the other

2. Define each of the variables in terms of the parameter $t$ to get parametric equations for the intersection curve

$x = r(t)_1$

$y = r(t)_2$

$z = r(t)_3$

3. Generate the vector function that describes the intersection curve using the formulas:

$r(t) = r(t)_1 \mathbf{i} + r(t)_2 \mathbf{j} + r(t)_3 \mathbf{k}$

$r(t) = \langle r(t)_1, r(t)_2, r(t)_3 \rangle$

Example:

Find the vector function for the curve of intersection of the following surfaces.

An ellipsoid $z = \sqrt{ 1 + x^2 - y^2 }$

A plane $z = 2 + x$

Since both curves have $z$ on the left side, we can set the right-hand sides equal to one another and solve for one variable of the remaining variables in terms of each other.

$\sqrt{ 1 + x^2 - y^2 } = 2 + x$

$1 + x^2 - y^2 = (2 + x)^2$

$1 + x^2 - y^2 = 4 + 4x + x^2$

$-3 - y^2 = 4x$

$x = - \frac{3}{4} - \frac{1}{4} y^2$

We want to define each variable in terms of the parameter $t$, so we'll set $y = t$.

$x = - \frac{3}{4} - \frac{1}{4} t^2$

To find $z$ in terms of $t$, we'll plug $x$ in terms of $t$ into $z = 2 + x$.

$z = 2 + x$

$z = 2 - \frac{3}{4} - \frac{1}{4}t^2$

$z = \frac{5}{4} - \frac{1}{4}t^2$

Now we have parametric equations for the curve of intersection, defined by:

$x = - \frac{3}{4} - \frac{1}{4}t^2$

$y = t$

$z = \frac{5}{4} - \frac{1}{4}t^2$

With the parametric equations in hand, we can plug each of them into the formula for the vector function.

$r(t) = r(t)_1 \mathbf{i} + r(t)_2 \mathbf{j} + r(t)_3 \mathbf{k}$

$r(t) = \left( \frac{3}{4} - \frac{1}{4}t^2 - \frac{1}{4}t^2 \right) \mathbf{i} + t \mathbf{j} + \left( \frac{5}{4} - \frac{1}{4}t^2 \right)$

This is the vector function for the curve of intersection, and can also be written:

$r(t) = \biggl \langle - \frac{3}{4} - \frac{1}{4}t^2, ~t, ~\frac{5}{4} - \frac{1}{4}t^2 \biggr \rangle$

Vectors - Derivatives and Integrals of Vector Functions

Derivative of a Vector Function

To find the derivative of a vector function, we simply find the derivatives of the coefficients when the vector function is in the form:

$r(t) = r(t)_1 \mathbf{i} + r(t)_2 \mathbf{j} + r(t)_3 \mathbf{k}$

If the vector is in the form $r(t) = \langle r(t)_1, r(t)_2, r(t)_3 \rangle$, it can first be transformed into the form above before taking the derivative.

$r'(t) = r'(t)_1 \mathbf{i} + r'(t)_2 \mathbf{j} + r'(t)_3 \mathbf{k}$

$r'(t) = \langle r'(t)_1, r(t)_2, r(t)_3 \rangle$

Example:

Find the derivative of the vector function:

$r(t) = \langle t^2, e^{3t}, t^3 ~sin(4t) \rangle$

The given function is the same as:

$r(t) = t^2 \mathbf{i} + e^{3t} \mathbf{j} + t^3 ~sin(4t) \mathbf{k}$

We'll use the product rule to find the derivative of the coefficient $\mathbf{k}$.

$r'(t) = 2t \mathbf + e^{3t}(3) \mathbf{j} + [3t^2 (sin(4t)) + t^3 (cos(4t)(4))] \mathbf{k}$

$r'(t) = 2t \mathbf{i} + 3e^{3t} \mathbf{j} + (3t^2 ~sin(4t) + 4t^3 ~cos(4t))$

or in the other form:

$r'(t) = \langle 2t, 3e^{3t}, 3t^2 ~sin(4t) + 4t^3 ~cos(4t) \rangle$

Unit Tangent Vector

To find the unit tangent vector for a vector function, we use the formula:

$T(t) = \frac{r'(t)}{||r'(t)||}$

where $r'(t)$ is the derivative of the vector function $r(t)$ and $t$ is given. $||r'(t)||$ is the magnitude of the derivative of the vector function at time $t$.

$||r'(t)|| = \sqrt{ [r'(t)_1]^2 + [r'(t)_2]^2 + [r'(t)_3]^2 }$

Example:

Find the unit tangent vector of the following vector function at $t=1$.

$r(t) = 4t^3 \mathbf{i} + 6t \mathbf{j} + 4t ~ln(t) \mathbf{k}$

We'll start by finding the derivative of the vector function.

$r'(t) = 12t^2 \mathbf{i} + 6 \mathbf{j} + \left[ (4)(ln(t)) + (4t) \left( \frac{1}{t} \right) \right] \mathbf{k}$

$r'(t) = 12t^2 \mathbf{i} + 6 \mathbf{j} + [4 ~ln(t) + 4] \mathbf{k}$

Now we'll find the derivative at $t=1$.

$r'(1) = 12(1)^2 \mathbf{i} + 6 \mathbf{j} + [4 ~ln(1) + 4] \mathbf{k}$

$r'(1) = 12 \mathbf{i} + 6 \mathbf{j} + [4(0) + 4] \mathbf{k}$

$r'(1) = 12 \mathbf{i} + 6 \mathbf{j} + 4 \mathbf{k}$

Now we'll use the values from the derivative to find the magnitude of the vector function at $t=1$ so that we can plug it into the formula for the unit tangent vector.

$||r'(t)|| = \sqrt{ [r'(t)_1]^2 + [r'(t)_2]^2 + [r'(t)_3]^2 }$

$||r'(t)|| = \sqrt{ 12^2 + 6^2 + 4^2 }$

$||r'(t)|| = \sqrt{144 + 36 + 16}$

$||r'(t)|| = \sqrt{196}$

$||r'(t)|| = 14$

Plugging everything into the formula for the unit tangent vector, we get:

$T(1) = \frac{ 12 \mathbf{i} + 6 \mathbf{j} + 4 \mathbf{k} }{14}$

$T(1) = \frac{12}{14} \mathbf{i} + \frac{6}{14} \mathbf{j} + \frac{4}{14} \mathbf{k}$

$T(1) = \frac{6}{7} \mathbf{i} + \frac{3}{7} \mathbf{j} + \frac{2}{7} \mathbf{k}$

This is the equation for the unit tangent vector.

Parametric Equations of the Tangent Line

The parametric equations of the tangent line of a vector function $r(t) = \langle r(t)_1, r(t)_2, r(t)_3 \rangle$ are:

$x = x_1 + r'(t_0)_1 t$

$y = y_1 + r'(t_0)_2 t$

$z = z_1 + r'(t_0)_3 t$

$x_1$, $y_1$, and $z_1$ come from the point $P(x_1, y_1, z_1)$ which is the point of tangency.

You find $r'(t)_1$, $r'(t)_2$, and $r'(t)_3$ by taking the derivative of the vector. You find $t_0$ by plugging $P(x_1, y_1, z_1)$ into the vector function. Then you find $r'(t_0)_1$, $r'(t_0)_2$, and $r'(t_0)_3$ by plugging $t_0$ into the derivative of the vector function.

Example:

Find the parametric equations of the following tangent line to the vector at point $P$.

$x = e^t$

$y = -t ~cos ~t$

$z = sin ~t$

at $P(1,0,0)$

Since the point of tangency is $P(1,0,0)$, we can plug that point into the formulas for the parametric equation of the tangent line.

$x = 1 + r'(t_0)_1 t$

$y = 0 + r'(t_0)_2 t$

$z = 0 + r'(t_0)_3 t$

Now we need to find a value for $t_0$. We'll use $x = e^t$, change $t$ to $t_0$, and plug in $x=1$ from $P(1,0,0)$ to get:

$1 = e^{t_0}$

$ln ~1 = ln ~e^{t_0}$

$t_0 = 0$

Plugging $t_0 = 0$ and $y=0$ into $y = -t ~cos ~t$ gives:

$0 = -0 ~cos ~0$

$0 = 0$

Now we'll plug $t_0 = 0$ and $z=0$ into $z = sin ~t$ and get:

$0 = sin ~0$

$0 = 0$

We've now shown that $t_0 = 0$ satisfies $x = e^t, y = -t ~cos ~t$ and $z = sin ~t$, so $0$ is the value we want to use for $t_0$. Therefore, the parametric equations of the tangent line become:

$x = 1 + r'(0)_1 t$

$y = r'(0)_2 t$

$z = r'(0)_3 t$

Next we need to find the derivative of the vector function. The original function is:

$r(t) = \langle e^t, -t ~cos ~t, sin ~t \rangle$

so its derivative is:

$r'(t) = \langle e^t, (-1)(cos ~t) + (-1)(-sin ~t), cos(t) \rangle$

$r'(t) = \langle e^t, - cos ~t + r ~sin ~t, cos ~t \rangle$

$r'(t) = \langle e^t, t ~sin ~t - cos ~t, cos ~t \rangle$

Plugging $t_0$ into the derivative, we get:

$r'(0) = \langle e^0, 0 ~sin ~0 - cos ~0, cos ~0 \rangle$

$r'(0) = \langle 1, 0-1, 1 \rangle$

$r'(0) = \langle 1, -1, 1 \rangle$

We plug these three values into our parametric equations, which become:

$x = 1 + 1t$

$y = -1t$

$z = 1t$

and these simplify to:

$x = 1+t$

$y = -t$

$z = t$

Integral of a Vector Function

To find the integral of a vector function, we simply replace each coefficient with its integral.

$\int r(t) ~dt = \mathbf{i} \int r(t)_1 ~dt + \mathbf{j} \int r(t)_2 ~dt + \mathbf{k} \int r(t)_3 ~dt$

If the vector function is given as $r(t) = \langle r(t)_1, r(t)_2, r(t)_3 \rangle$, then its integral is:

$\int r(t) = \langle \int r(t)_1 ~dt, \int r(t)_2 ~dt, \int r(t)_3 ~dt \rangle$

Example:

Find the integral of the following vector function over the interval $[0, \pi]$.

$r(t) = sin(2t) \mathbf{i} + 2e^{2t} \mathbf{j} + 4t^3 \mathbf{k}$

$\int_0^{\pi} r(t) ~dt = \frac{-cos(2t)}{2} \bigg|_0^{\pi} \mathbf{i} + \frac{2e^{2t}}{2} \bigg|_0^{\pi} \mathbf{j} + \frac{4t^4}{4} \bigg|_0^{\pi} \mathbf{k}$

$\int_0^{\pi} r(t) ~dt = \frac{-cos(2t)}{2} \bigg|_0^{\pi} \mathbf{i} + e^{2t} \bigg|_0^{\pi} \mathbf{j} + t^4 \bigg|_0^{\pi} \mathbf{k}$

Evaluating over the interval $[0, \pi]$, we get:

$\int_0^{\pi} r(t) ~dt = \left[ \frac{-cos(2 \pi)}{2} - \frac{-cos(2(0))}{2} \right] \mathbf{i} + [e^{2 \pi} - e^2(0)] \mathbf{j} + [\pi^4 - 0^4] \mathbf{k}$

$\int_0^{\pi} r(t) ~dt = \left[ \frac{-cos(2 \pi)}{2} + \frac{cos ~0}{2} \right] \mathbf{i} + (e^{2 \pi} - 1) \mathbf{j} + (\pi^4 - 0) \mathbf{k}$

$\int_0^{\pi} r(t) ~dt = \left( \frac{-1}{2} + \frac{1}{2} \right) \mathbf{i} + (e^{2 \pi} - 1) \mathbf{j} + \pi^4 \mathbf{k}$

$\int_0^{\pi} r(t) ~dt = 0 \mathbf{i} + (e^{2 \pi} - 1) \mathbf{j} + \pi^4 \mathbf{k}$

$\int_0^{\pi} r(t) ~dt = (e^{2 \pi}) \mathbf{j} + \pi^4 \mathbf{k}$

This is the integral of the vector function. We could also write it in the form:

$\int_0^{\pi} r(t) ~dt = \langle 0, e^{2 \pi} - 1, \pi^4 \rangle$

Vectors - Arc Length and Curvature

Arc Length of a Vector Function

To find the arc length of a vector function, we use the formula:

$ L = \int_1^b \sqrt{ \left( \frac{dx}{dt} \right)^2 + \left( \frac{dy}{dt} \right)^2 + \left( \frac{dz}{dt} \right)^2 } ~dt $

  • $L$ is the arc length of the vector function
  • $[a,b]$ is the interval that defines the arc
  • $\frac{dx}{dt}$, $\frac{dy}{dt}$, and $\frac{dz}{dt}$ are the derivatives of the parametric equations of $x$, $y$, and $z$. The parametric equations are:
    • $x = r(t)_1$
    • $y = r(t)_2$
    • $z = r(t)_3$

Example:

Find the arc length of the vector function over the given interval:

$r(t) = \langle sin(2t), cos(2t), 2t \rangle$

on $0 \le t \le 2$

We'll pull the parametric equations out of the vector function as:

$x = sin(2t)$

$y = cos(2t)$

$z = 2t$

Now take the derivative of each.

$\frac{dx}{dt} = 2 ~cos(2t)$

$\frac{dy}{dt} = -2 ~sin(2t)$

$\frac{dz}{dt} = 2$

Plugging the derivatives and the given interval $0 \le t \le 2$ into the formula for arc length, we get:

$L = \int_0^2 \sqrt{ [2 ~cos(2t)]^2 + [-2 ~sin(2t)]^2 + (2)^2 } ~dt$

$L = \int_0^2 \sqrt{ 4 ~cos^2(2t) + 4 ~sin^2(2t) + 4 } ~dt$

$L = \int_0^2 \sqrt{ 4 ~cos^2(2t) + sin^2(2t) + 4 } ~dt$

Since $cos^2x + sin^2x = 1$, we can simplify the integral to:

$L = \int_0^2 \sqrt{4(1) + 4} ~dt$

$L = \int_0^2 \sqrt{8} ~dt$

$L = \int_0^2 \sqrt{4 \cdot 2} ~dt$

$L = \int_0^2 2 \sqrt{2} ~dt$

$L = 2 \sqrt{2} t \bigg|_0^2$

Evaluating over the interval, we get:

$L = 2 \sqrt{2}(2) - 2 \sqrt{2}(0)$

$L = 4 \sqrt{2}$

The arc length of the vector function over the interval $0 \le t \le 2$ is $L = 4 \sqrt{2}$.

Reparametrizing the Curve

When we reparametrize a curve, it means that we rewrite it in terms of an indpendent variable. It is common to use the letter $s$ for this variable. Given a vector function, $r(t) = r(t)_1 \mathbf{i} + r(t)_2 \mathbf{j} + r(t)_3 \mathbf{k}$, the reparametrized curve will be:

$r[t(s)] = t[t(s)]_1 \mathbf{i} + r[t(s)]_2 \mathbf{j} + r[t(s)]_3 \mathbf{k}$

To reparametrize the curve with respect to arc length, we'll have to find the arc length of the vector function using:

$ L = \int_a^b \sqrt{ \left( \frac{dx}{dt}^2 \right) + \left( \frac{dy}{dt}^2 \right) + \left( \frac{dz}{dt}^2 \right) } $

where $L$ is the arc length of the function, and [a,b] is the interval to evaluate upon. The derivatives are with respect to the parametric equations for $x$, $y$, and $z$. Regardless of how the vector function is expressed, the parametric equations are:

$x = r(t)_1$

$y = r(t)_2$

$z = r(t)_3$

Example:

Reparametrize the following curve with respect to arc length from $t=0$ in the direction of increasing $t$.

$r(t) = -2t \mathbf{i} + 4t \mathbf{j} + (2-4t) \mathbf{k}$

First, we'll use the coefficients from the vector function to generate parametric equations of the vector function.

$x = -2t$

$y = 4t$

$z = 2 - 4t$

Now take the derivatives of these.

$\frac{dx}{dt} = -2$

$\frac{dy}{dt} = 4$

$\frac{dz}{dt} = -4$

Next, we'll plug the derivatives into the arc length formula. The limits of integration will be $[0,t]$, since the problem indicated that the arc length should start at $0$ and head in the direction of increasing $t$.

$L = \int_0^t \sqrt{ (-2)^2 + (4)^2 + (-4)^2 } ~dt$

$L = \int_0^t \sqrt{4 + 16 + 16} ~dt$

$L = \int_0^t \sqrt{36} ~dt$

$L = \int_0^t 6 ~dt$

$L = 6t \bigg|_0^t$

$L = 6(t) - 6(0)$

$L = 6t$

Now we'll set arc length equal to the independent variable $s$, and get:

$s = $6t

Then solve for $t$.

$t = \frac{s}{6}$

We'll substitute the value of t into the original vector function, and get the reparametrized version.

$ r[t(s)] = -2 \left( \frac{5}{6} \right) \mathbf{i} + 4 \left( \frac{5}{6} \right) \mathbf{j} + \left[ 2-4 \left( \frac{5}{6} \right) \right] \mathbf{k} $

$r[t(s)] = - \frac{1}{3}s \mathbf{i} + \frac{2}{3}s \mathbf{j} + \left( 2 - \frac{2}{3}s \right) \mathbf{k}$

This is the reparametrized curve in terms of arc length.

Unit Tangent and Unit Normal Vectors

The unit tangent vector $T(t)$ of a vector function $r(t) = r(t)_1 \mathbf{i} + r(t)_2 \mathbf{j} + r(t)_3 \mathbf{k}$ is the vector that is $1$ unit long and tangent to the vector function at the point $t$.

$T(t) = \frac{ r'(t) }{ |f'(t)| }$

$|r'(t)|$ is the magnitude of the derivative of the vector function at time $t$, found as:

$|r'(t)| = \sqrt{ [r'(t)_1]^2 + [r'(t)_2]^2 + [r'(t)_3]^2 }$

The unit normal vector $N(t)$ of the same vector function is the vector that is $1$ unit long and perpendicular to the unit tangent vector at the same point $t$.

$N(t) = \frac{ T'(t) }{ |T'(t) }$

$|T'(t)|$ is the magnitude of the derivative of the unit tangent vector at time $t$.

$|T'(t)| = \sqrt{ [T'(t)_1]^2 + [T'(t)_2]^2 + [T'(t)_3]^2 }$

Example:

Find the unit normal vector of the following vector function at $t=1$.

$r(t) = t \mathbf{i} + t^2 \mathbf{j} + 2 \mathbf{k}$

We must first take the derivative of the vector function to find $r'(t)$.

$r(t) = t \mathbf{i} + t^2 \mathbf{j} + 2 \mathbf{k}$

$r'(t) = 1 \mathbf{i} + 2t \mathbf{j} + 0 \mathbf{k}$

$r'(t) = \mathbf{i} + 2t \mathbf{j}$

Now we can use $r'(t)$ to find $|r'(t)|$.

$|r'(t)| = \sqrt{ (1)^2 + (2t)^2 + (0)^2 }$

$|r'(t)| = \sqrt{1 + 4t^2}$

Plugging these into the formula for the unit tangent vector, we get:

$T(t) = \frac{ r'(t) }{ |f'(t)| }$

$T(t) = \frac{ \mathbf{i} + 2t \mathbf{j} }{ \sqrt{ 1 + 4t^2 } }$

Now that we have an equation for the unit tangent vector, we can use it to find the unit normal vector, given by:

$N(t) = \frac{ T'(t) }{ |T'(t)| }$

We'll have to start by taking the derivative of the unit tangent vector to get $T'(t)$, using the quotient rule.

$T(t) = \frac{ \mathbf{i} + 2t \mathbf{j} }{ \sqrt{1 + 4t^2} }$

$ T'(t) = \frac { (0 \mathbf{i} + 2 \mathbf{j}) \sqrt{1 + 4t^2} - (\mathbf{i} + 2t \mathbf{j}) \frac{1}{2} (1 + 4t^2)^{-\frac{1}{2}} (8t) } { 1 + 4t^2 } $

$ T'(t) = \frac { 2 \mathbf{j} \sqrt{1 + 4t^2} - 4t(\mathbf{i} + 2t \mathbf{j}) (1 + 4t^2)^{-frac{1}{2}} } { 1 + 4t^2 } $

$ T'(t) = \sqrt{ \frac { 2 \mathbf{j} \sqrt{1 + 4t^2} - \frac{ 4t(\mathbf{i} + 2t \mathbf{j}) }{ \sqrt{1 + 4t^2} } } { 1 + 4t^2 } } $

$ T'(t) = \frac { \frac{ 2 \mathbf{j} \sqrt{1+4t^2} \sqrt{1+4t^2} }{ \sqrt{1 + 4t^2} } - \frac{ 4t(1 + 2t \mathbf{j}) }{ \sqrt{1 + 4t^2} } } { 1 + 4t^2 } $

$ T'(t) = \frac { \frac{ 2 \mathbf{j} (1 + 4t^2) }{ \sqrt{1 + 4t^2} } - \frac{ 4t(\mathbf{i} + 2t \mathbf{j}) }{ \sqrt{1 + 4t^2} } } { 1 + 4t^2 } $

$ T'(t) = \frac { \frac{ 2 \mathbf{j}(1 + 4t^2) - 4t(\mathbf{i} + 2t \mathbf{j}) }{ \sqrt{1 + 4t^2} } } { 1 + 4t^2 } $

$T'(t) = \frac{ 2 \mathbf{j}(1 + 4t^2) - 4t(\mathbf{i} + 2t \mathbf{j}) }{ (1 + 4t^2)^{\frac{3}{2}} }$

$T'(t) = \frac{ 2 \mathbf{j} + 8t^2 \mathbf{j} - 4t \mathbf{i} - 8t^2 \mathbf{j} }{ (1 + 4t^2)^\frac{3}{2} }$

$T'(t) = \frac{ -4t \mathbf{i} + 2 \mathbf{j} }{ (1 + 4t^2)^\frac{3}{2} }$

Next, we'll solve for $T'(1)$ by plugging $t=1$ into the derivative we just found.

$T'(1) = \frac{ -4(1) \mathbf{i} + 2 \mathbf{j} }{ (1 + 4t^2)^{\frac{3}{2}} }$

$T'(1) = \frac{ -4 \mathbf{i} + 2 \mathbf{j} }{ 5^{\frac{3}{2}} }$

$T'(1) = - \frac{4}{5 \sqrt{5}} \mathbf{i} + \frac{2}{5 \sqrt{5}} \mathbf{j}$

Now we'll find the magnitude $|T'(t)|$.

$|T'(1)| = \sqrt{ \left( - \frac{4}{5 \sqrt{5}} \right)^2 + \left( \frac{2}{5 \sqrt{5}} \right)^2 + (0)^2 }$

$|T'(1)| = \sqrt{ \frac{16}{25(5)} + \frac{4}{25(5)} }$

$|T'(1)| = \sqrt{ \frac{20}{125} }$

$|T'(1)| = \sqrt{ \frac{4}{25} }$

$|T'(1)| = \frac{2}{5}$

Finally, we'll plug everything we've found into the formula for the unit normal vector.

$N(t) = \frac{T'(t)}{|T'(t)|}$

$ N(1) = \frac { - \frac{4}{5 \sqrt{5}} \mathbf{i} + \frac{2}{5 \sqrt{5}} \mathbf{j} } { \frac{2}{5} } $

$N(1) = - \frac{4(5)}{5(2) \sqrt{5}} \mathbf{i} + \frac{2(5)}{5(2) \sqrt{5}} \mathbf{j}$

$N(1) = - \frac{2}{\sqrt{5}} \mathbf{i} + \frac{1}{\sqrt{5}} \mathbf{j}$

We can multiply by $\sqrt{5}$ to simplify the equation of the unit normal vector to:

$N(1) = -2 \mathbf{i} + \mathbf{j}$

This is the unit normal vector of the vector function $r(t) = t \mathbf{i} + t^2 \mathbf{j} + 2 \mathbf{k}$ at the point $t=1$.

Curvature

To find the curvature $k(t)$ of a vector function $r(t) = r(t)_1 \mathbf{i} + r(t)_2 \mathbf{j} + r(t)_3 \mathbf{k}$, we'll use the equation:

$k(t) = \frac{ |T'(t)| }{ |r'(t)| }$

$|t'(t)|$ is the magnitude of the unit tangent vector $T(t)$. To find $k(t)$, we will:

  1. Find $r'(t)$, and use it to:
  2. Find $|r'(t)|$ and then use $r'(t)$ and $|r'(t)|$ to:
  3. Find $T(t)$, and then use it to:
  4. Find $T'(t)$, and then use it to:
  5. Find $|T'(t)|$, and use $|r'(t)|$ and $|T'(t)|$ to:
  6. Find $k(t)$

Example:

$r(t) = 4t \mathbf{i} + t^2 \mathbf{j} + 2t \mathbf{k}$

We'll start by calculating the derivative of the vector function:

$r'(t) = 4 \mathbf{i} + 2t \mathbf{j} + 2t \mathbf{k}$

We'll start by calculating the derivative of the vector function:

$r'(t) = 4 \mathbf{i} + 2t \mathbf{j} + 2 \mathbf{k}$

Then we'll find $|r'(t)|$

$|r'(t)| = \sqrt{ (4)^2 + (2t)^2 + (z)^2 }$

$|r'(t)| = \sqrt{ (4)^2 + (2t)^2 + (2)^2 }$

$|r'(t)| = \sqrt{ 16 + 4t^2 + 4 }$

$|r'(t)| = \sqrt{ 4t^2 + 20 }$

$|r'(t)| = \sqrt{ 4(t^2 + 5) }$

$|r'(t)| = 2 \sqrt{t^2 + 5}$

Now we'll use the derivative and its magnitude to find an equation for the unit tangent vector $T(t)$.

$T(t) = \frac{ f'(t) }{ |r'(t)| }$

$T(t) = \sqrt{ 4 \mathbf{i} + 2t \mathbf{k} + 2 \mathbf{k} }{ 2 \sqrt{ t^2 + 5 } }$

$ T(t) = \frac{4}{2 \sqrt{t^2 + 5}} \mathbf{i} + \frac{2t}{2 \sqrt{t^2 + 5}} \mathbf{j} + \frac{2}{s \sqrt{t^2 + 5}} \mathbf{k} $

$ T'(t) = \frac{2}{t^2 + 5} \mathbf{i} + \frac{t}{\sqrt{t^2 + 5}} \mathbf{j} + \frac{1}{\sqrt{t^2 + 5}} \mathbf{k} $

Now we can find the derivative of the unit tangent vector $T'(t)$, using the quotient rule.

$ T'(t) = \frac{ (0) \sqrt{t^2 + 5} - (2) \left[ \frac{1}{2} (t^2 + 5)^{- \frac{1}{2}} (2t) \right]} {(t^2+5)^2} \mathbf{i} + \frac{ (1) \sqrt{t^2 + 5} - (t) \left[ \frac{1}{2} (t^2 + 5)^{- \frac{1}{2}} (2t) \right]} {(t^2+5)^2} \mathbf{j} + \frac{ (0) \sqrt{t^2 + 5} - (1) \left[ \frac{1}{2} (t^2 + 5)^{- \frac{1}{2}} (2t) \right]} {(t^2+5)^2} \mathbf{k} $

$ T'(t) = \frac{ -2t(t^2 + 5)^{- \frac{1}{2}} }{ t^2 + 5 } \mathbf{i} + \frac{ \sqrt{t^2 - 5} - t^2(t^2 + 5)^{- \frac{1}{2}} }{ t^2 + 5 } \mathbf{j} + \frac{ -t(t^2 + 5)^{- \frac{1}{2}} }{ t^2 + 5 } \mathbf{k} $

$ T'(t) = - \frac{ 2t }{ (t^2 + 5)^\frac{3}{2} } \mathbf{i} + \frac{ \sqrt{t^2+5} - \frac{t^2}{t^2+5} }{ t^2 + 5 } \mathbf{j} - \frac{ t }{ (t^2 + 5)^{\frac{3}{2}} } \mathbf{k} $

$ T'(t) = - \frac{ 2t }{ (t^2 + 5)^{\frac{3}{2}} } \mathbf{i} + \frac{ \frac{t^2+5}{\sqrt{t^2+5}} - \frac{t^2}{\sqrt{t^2+5}} }{ t^2 + 5 } \mathbf{j} - \frac{ t }{ (t^2+5)^\frac{3}{2} } \mathbf{k} $

$ T'(t) = - \frac{ 2t }{ (t^2 + 5)^{\frac{3}{2}} } \mathbf{i} + \frac{ \frac{t^2 + 5 - t^2}{\sqrt{t^2+5}} }{ t^2 + 5 } \mathbf{j} - \frac{ t }{ (t^2+5)^\frac{3}{2} } \mathbf{k} $

$ T'(t) = - \frac{ 2t }{ (t^2 + 5)^{\frac{3}{2}} } \mathbf{i} + \frac{ \frac{5}{\sqrt{t^2+5}} }{ t^2 + 5 } \mathbf{j} - \frac{ t }{ (t^2+5)^\frac{3}{2} } \mathbf{k} $

$ T'(t) = - \frac{2t}{(t^2+5)^\frac{3}{2}} \mathbf{i} + \frac{5}{(t^2+5)^\frac{3}{2}} \mathbf{j} - \frac{t}{(t^2+5)^\frac{3}{2}} \mathbf{k} $

Then we'll find the magnitude of the derivative of the unit tangent vector.

$|T'(t)| = \sqrt{ [T'(t)_1]^2 + [T'(t)_2]^2 + [T'(t)_3]^2 }$

$ |T'(t)| = \sqrt{ \left[ - \frac{2t}{ (t^2 + 5)^{\frac{3}{2}} } \right]^2 + \left[ \frac{5}{ (t^2+5)^\frac{3}{2} } \right]^2 + \left[ - \frac{t}{(t^2 + 5)^\frac{3}{2}} \right]^2 } $

$|T'(t)| = \sqrt{ \frac{4t^2}{(t^2 + 5)^3} + \frac{25}{(t^2 + 5)^3} + \frac{t^2}{(t^2 + 5)^3} }$

$|T'(t)| = \sqrt{ \frac{5t^2 + 25}{(t^2 + 5)^3} }$

$|T'(t)| = \sqrt{ \frac{5(t^2 + 5)}{(t^2 + 5)^3} }$

$|T'(t)| = \sqrt{ \frac{5}{(t^2 + 5)^2} }$

$|T'(t)| = \frac{\sqrt{5}}{t^2 + 5}$

Finally, we can solve for the curvature of the function $k(t)$.

$k(t) = \frac{ |T'(t)| }{ |r'(t)| }$

$k(t) = \frac{ \frac{\sqrt{5}}{t^2+5} }{ 2 \sqrt{t^2 + 5} }$

$k(t) = \frac{ \sqrt{5} }{ 2(t^2 + 5)^\frac{3}{2} }$

This is the curvature of the vector function.

Maximum Curvature

Before we can find maximum curvature of a vector function, we must find the curvature $k(t)$. Once found, we take its derivative, set it to $0$, and solve for $t$. If there is only one value for $t$, that value is associated with the maximum curvature. If there is more than one value for $t$, we'll use the second derivative test to determine which one represents maximum curvature.

Example:

Find the maximum curvature of the following vector function with the given curvature:

$k(t) = 8t^2 - 4t$

First, we'll find the derivative of k(t).

$k'(t) = 16t - 4$

Next, we'll set $k'(t)$ to $0$ and solve for $t$.

$0 = 16t - 4$

$-16t = -4$

$t = \frac{-4}{-16}$

$t = \frac{1}{4}$

The maximum curvature occurs when $t = \frac{1}{4}$.
