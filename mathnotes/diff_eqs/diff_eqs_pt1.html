First Order Equations

In the simplest sense, a differential equation is just an equation that contains one or more derivatives. For example, $y' - y = 3t^2$. First order equations include a first derivative as the highest-order derivative, and second order equations include a second derivative as the highest-order derivative, etc.

Classifying Differential Equations

The partial derivative of a function is the derivative of that function with respect to one of the multiple variables in which the function is defined. If a function $f$ has two variables $x$ and $y$, then we know that it has two partial derviatives, $\frac{\partial f}{\partial x}$ and $\frac{\partial f}{\partial y}$.

We cannot name only one derivative for $f$, because $f$ is defined in two variables. Instead, we find one partial derivative of $f$ for each of its variables. When we define equations using partial derivatives like these, we call them partial differential equations (PDEs).

Ordinary Derivatives

An ordinary derivative is the derivative of a function in a single variable. For example, given a function for $y$ in terms of $x$, $y(x)$, its first derivative can be written as $\frac{dy}{dx}$ (without the partial symbol).

Linear Differential Equations

Linear differential equations are given in the form:

$p_n(x)t^{(n)}(x) + p_{n-1}(x)y^{(n-1)}(x) + \ldots + p_1(x)y'(x) + p_0(x)y(x) = q(x)$

where $p_i(x)$ and $q(x)$ are functions of $x$. If a first order ODE doesn't match this form, we say that it is a non-linear equation. Key elements of this definition of linear equations are:

  1. All fo the $p_i(x)$ coefficients are functions in terms of only $x$
  2. $q(x)$ is also a function in terms of only $x$
  3. The power of $y$ is never higher than $1$ (we should only see $y$ and its derivatives)

For example, the equation $y' - 2xy = x ~cos ~y$ is non-linear because $q(x) = x ~cos ~y$, which means it is defined in terms of both $x$ and $y$, not $x$ alone.

When $q(x) = 0$, we say the linear equation is homogenous. When $q(x) \neq 0$, non-homogenous.

Linear Equations

First order linear differential equations are given in the form $\frac{dy}{dx} + P(x)y = Q(x)$, where $P(x)$ and $Q(x)$ are continuous functions. They could be in a different form, such as $P(x)y = Q(x) - \frac{dy}{dx}$.

When $Q(x) = 0$, the first order linear equation is homogenous and the equation simplifies to $\frac{dy}{dx} + P(x)y = 0$. These homogenous linear equations are also called seperable differential equations. The solution to a homogenous linear equation is: $y = Ce^{K(x)}$, where $K(x) = - \int P(x) ~dx$.

Solving Equations with The Integrating Factor

To solve a first order linear differential equation, we can always follow the same steps:

  1. Make sure the equation matches the general form of a first order linear differential equation
  2. From the equation, identify $P(x)$ and $Q(x)$
  3. Find the equation's integrating factor, $I(x) = e^{ \int P(x) ~dx }$
  4. Multiply through the linear equation by the integrating factor
  5. Reverse the product rule to rewrite the left side of the resulting equation as $\left( \frac{d}{dx} \right) [yI(x)]$
  6. Integrate both sides of the equation to find $y = y(x)$

Example:

Find the solution to the following linear differential equation:

$x \frac{dy}{dx} - 2y = x^2$

Step 1 is to match the given equation to the standard form of a first order linear differential equation, by dividing both sides of the equation by $x$.

$\frac{dy}{dx} - \frac{2}{x}y = x$

Step 2 is to identify $P(x)$ and $Q(x)$:

$P(x) = - \frac{2}{x}$

$Q(x) = x$

Step 3, now that we have $P(x)$, is to find the integrating factor:

$I(x) = e^{\int P(x) ~dx}$

$I(x) = e^{\int \frac{-2}{x} ~dx}$

$I(x) = e^{-2 ~ln ~x}$

$I(x) = e^{ln ~x^{-2}}$

$I(x) = \frac{1}{x^2}$

Notice that we didn't add $C$ to account for the constant of integration. We can leave it out, because we're only looking for one solution to the linear differential equation, and that one solution can be associated with $C=0$.

Step 4 is to multiply both sides of the equation by the integrating factor.

$\frac{dy}{dx} \left( \frac{1}{x^2} \right) - \frac{2}{x}y \left( \frac{1}{x^2} \right) = x \left( \frac{1}{x^2} \right)$

$\frac{dy}{dx} \left( \frac{1}{x^2} \right) - \frac{2}{x^3}y = \frac{1}{x}$

Rewriting $\frac{dy}{dx}$ as $y'$, we get:

$y'x^{-2} - 2yx^{-3} = x^{-1}$

The left side of $y'x^{-2} - 2yx^{-3} = x^{-1}$ is actually the derivative of $yx^{-2}$. This is interesting because $yx^{-2}$ is the product of $y$ and our integrating factors. When we multiply our original linear differential equation by its integrating factor, the left side of the resulting equation will always be equal to $\frac{d}{dx}(yI(x))$.

Step 5: we make a substitution, replacing the left side of the resulting equation:

$\frac{d}{dx}(yx^{-2}) = x^{-1}$

Step 6: on the left, the derivative and the integral cancel out, since they are inverse operations.

$\int \frac{d}{dx}(yx^{-2}) = \int x^{-1} ~dx$

$yx^{-2} = ln |x| + C$

We solve for $y$, and find the solution to the linear differential equation

$y = x^2(ln ~|x| + C)$

Example:

Solve the following linear differential equation:

$\frac{dy}{dx} + 2y = 4e^{-2x}$

The equation is already in standard form for a linear differential equation, so we can identify $P(x) = 2$ and $Q(x) = 4e^{-2x}$, and use $P(x)$ to find the integrating factor.

$P(x) = e^{ \int P(x) ~dx }$

$P(x) = e^{ \int 2 ~dx }$

$P(x) = e^{2x}$

Multiply through the linear differential equation by the integrating factor:

$\frac{dy}{dx}(e^{2x}) + 2y(e^{2x}) = 4e^{-2x} (e^{2x})$

$\frac{dy}{dx} e^{2x} + 2e^{2x} y = 4e^0$

$\frac{dy}{dx} e^{2x} + 2e^{2x} y = 4$

Replace the left side of the equation with $\left( \frac{d}{dx} \right) (yI(x))$

$\frac{d}{dx}(ye^{2x}) = 4$

Then integrate both sides of the equation, since the integral and derivative will cancel out.

$\int \frac{d}{dx} (ye^{2x}) ~dx = \int 4 ~dx$

$ye^{2x} = 4x + C$

Dividing both sides by $e^{2x}$ to get $y$ by itself, the solution to the linear differential equation is:

$y = \frac{4x + C}{e^{2x}}$

Initial Value Problems

We are often interested in finding $C$, a constant of integration. The information we'll need to find $C$ is an initial condition, which is the value of the solution at a specific point. An initial condition for a first order differential equation will take the form $y(x_0) = y_0$, and the number of initial conditions required for a given differential equation will depend on the order of the differential equation.

An initial value problem is a differential equations problem in which we're asked to use some given initial condition, or set of conditions, in order to find the particular solution.

Solving Initial Value Problems:

  1. Find the general solution that contains the constant of integration $C$
  2. Substitute the initial condition $x = x_0$ and $y = y_0$ into the general solution to find the associated value of $C$
  3. Restate the general solution, and include the value $y(0) = -5$ in order to find the particular solution to the differential equation $\frac{dy}{dx} + 2y = 4e^{-2x}$.

In the previous lesson, we used the integrating factor to find the general solution to this differential equation, and it was:

$y = \frac{4x+C}{e^{2x}}$

Once we have this general solution, we recognize from the initial condition $y(0) = -5$ that $x=0$ and $y=-5$, so we plug these values into the general solution:

$-5 = \frac{4(0) + C}{e^{2(0)}}$

and then simplify this to solve for $C$:

$-5 = \frac{0+C}{1}$

$C = -5$

So the particular solution to the differential equation is:

$y = \frac{4x - 5}{e^{2x}}$

The general solution always remains the same, but the particular solution changes based on the initial condition we use.

Separable Equations

We saw that first order linear equations are differential equations in the form:

$\frac{dy}{dx} + P(x)y = Q(x)$

In contrast, first order separable differential equations have the form:

$N(y) \frac{dy}{dx} = M(x)$

$N(y)y' = M(x)$

We call these separable because we can separate the variables, the $x$ and $y$ terms, onto opposite sides of the equation with no mixing.

When working with separable differential equations, we usually prefer the Leibniz notation $\frac{dy}{dx}$, because it makes it easier to see how to separate variables. We move the $dx$ to the right, leaving just the $dy$ notation on the left.

$N(y) ~dy = M(x) ~dx$

How to Solve Separable Equations

Most often, we'll be given questions in the form:

$N(y) \frac{dy}{dx} = M(x)$

There is a standard process to solving. First, we'll separate variables by moving the $dx$ to the right side.

$N(y) ~dy = M(x) ~dx$

With variables separated, we'll integrate both sides of the equation.

$\int N(y) ~dy = \int M(x) ~dx$

Normally, when we evaluate an indefinite integral, we need to add in a constant of integration. So after integrating, we would expect to have something like:

$\mathbf{N}(y) + C_1 = \mathbf{M}(x) + C_2$

where $\mathbf{N}(y)$ is the integral of $N(y)$, and $\mathbf{M}(x)$ is the integral of $M(x)$. When we solve for $\mathbf{N}(y)$, we get $\mathbf{N}(y) = \mathbf{M}(x) + C_2 - C_1$. Both $C_2$ and $C_1$ are constants, so $C_2 - C_1$ is also a constant. We can substitute and represent $C_2 - C_1$ as $C$.

$\mathbf{N}(y) = \mathbf{M}(x) + C$

Once the equation is in this form, the goal is to express $y$ explicitly as a function of $x$. If we cannot, we will settle for an implicit function.

To summarize, our solution steps are:

  1. If necessary, rewrite the equation in Leibniz form
  2. Separate variables with $y$ terms on the left and $x$ terms on the right
  3. Integrate both sides of the equation, adding $C$ to the right side
  4. If possible, solve the solution equation specifically for $y$

Example:

Find the solution to the separable differential equation $y' = y^2 sin ~x$.

Separate variables:

$dy = y^2 sin ~x ~dx$

$\frac{1}{y^2} ~dy = sin ~x ~dx$

Integrating both sides, we get:

$\int \frac{1}{y^2} ~dy = \int sin ~x ~dx$

$\int y^{-2} ~dy = \int sin ~x ~dx$

$-y^{-1} = -cos ~x + C$

$- \frac{1}{y} = - cos ~x + C$

$\frac{1}{y} = cos ~x + C$

Finally, solving for $y$ gives the solution to the separable differential equation

$1 = y(cos ~x + C)$

$y = \frac{1}{cos ~x + C}$

Substitutions

First order linear and separable differential equations always have a solution, but first order differential equations that aren't linear or separable may or may not have one. To determine whether or not a non-linear, non-separable equation has a solution, one approach is to try transforming the differential equation into a linear or separable equation.

The substitution we want to make, also called a change of variable, is the one that allows us to convert the differential equation into a linear or separable equation.

Example:

Use a change of variable to solve the differential equation:

$y' = 2x + y$

If we choose the substitution $u = ax + by$, we can set up the substitution:

$u = 2x + y$

$u' = 2 + y$

We solve this second equation for $u' = 2 + y'$ to get $y' = u' - 2$, then substitute $y' = u' - 2$ into the left side of the ODE, and $u = 2x + y$ into the right side.

$u' - 2 = u$

Now the equation is separable.

$\frac{du}{dx} = u + 2$

$du = (u + 2) ~dx$

$\frac{1}{u+2} ~du = ~dx$

Then we integrate both sides:

$\int\frac{1}{u+2} ~du = \int ~dx$

$ln(u+2) = x + C$

$e^{ln ~u + 2} = e^{x + C}$

$u + 2 = Ce^{Mx} \to Ce^x$

$u = Ce^x - 2$

We'll back-substitute using $u = 2x + y$ and then solve for $y$ to find the solution to the differential equation.

$2x + y = Ce^x - 2$

$y = Ce^x - 2x - 2$

Example:

Use a change of variable to solve the differential equation:

$y' = (4x - y - 1)^2 = 0$

If we choose the substitution $u = ax + by$, we can set up the substitution $u = 4x - y$, and $u' = 4 - y'$. And then solve this second equation $u' = 4 - y'$ for $y'$ to get $y' = 4 - u'$. Then we'll substitute $y' = 4 - u'$ and $u = 4x - y$ into the left side of the ODE.

$4 - u' - (u - 1)^2 = 0$

$u' = 4 - (u - 1)^2$

Now the equation is separable, so we'll substitute variables.

$\frac{dy}{dx} = 4 - (u - 1)^2$

$du = (4 - (u - 1)^2) ~dx$

$\frac{du}{4-(u-1)^2} = ~dx$

And then integrate both sides:

$\int \frac{du}{4 - (u - 1)^2} = \int ~dx$

$\int \frac{du}{4 - (u^2 - 2u + 1)} = \int ~dx$

$\int \frac{du}{4 - u^2 + 2u - 1} = \int ~dx$

$\int \frac{du}{3 + 2u - u^2} = \int ~dx$

$- \int \frac{du}{u^2 - 2u - 3} = \int ~dx$

$- \int \frac{du}{(u-3)(u+1)} = \int ~dx$

Use partial functions to rewrite the integral on the left:

$\frac{1}{4} \int \frac{1}{u+1} - \frac{1}{u-3} = \int ~dx$

then evaluate the integrals on both sides.

$\frac{1}{4} (ln |u+1| - ln|u-3|) = x + C$

$ln \left| \frac{u+1}{u-3} \right| = 4x + C$

Raise both sides to the base $e$, then solve for $u$.

$\frac{u+1}{u-3} = Ce^{4x}$

$u + 1 = (u - 3)Ce^{4x}$

$u + 1 = uCe^{4x} - 3Ce^{4x}$

$u(1 - Ce^{4x}) = - (1 + 3Ce^{4x})$

$u = \frac{1 + 3Ce^{4x}}{1 - Ce^{4x}}$

$4x - y = - \frac{1 + 3Ce^{4x}}{1 - Ce^{4x}}$

$y = 4x + \frac{1 + 3Ce^{4x}}{1 - Ce^{4x}}$

Bernoulli Equations

A Bernoulli differential equation is a differential equation in the form $y' + p(x)y = q(x)y^n$, where $p(x)$ and $q(x)$ are continuous and $u$ is any real number. We always follow the same approach, which will be a change of variables in order to turn the Bernoulli equation into a linear equation.

If $n = 0$, the Bernoulli equation is already a first order linear equation.

$y + p(x)y = q(x)y^0 = q(x)$

If $n = 1$, the Bernoulli equation is already a first order separable differential equation.

$y' + p(x)y = q(x)y^1$

$y' + p(x)y - q(x)y = 0$

$y' + (p(x) - q(x))y = 0$

$\frac{dy}{dx} + (p(x) - q(x))y = 0$

$\frac{dy}{dx} = -(p(x) - q(x))y$

$dy = -(p(x) - q(x))y ~dx$

$\frac{1}{y} ~dy = - (p(x) - q(x)) ~dx$

$\frac{1}{y} ~dy = (q(x) - p(x)) ~dx$

For any value $n \neq 0$ or $1$, our goal will be to perform a change of variables. In general, these are the steps we'll follow:

  1. If the Bernoulli equation isn't already given in standard form, rewrite it in standard form
  2. Divide both sides of the equation by $y^n$
  3. Identify the substitution $v = y^{1-n}$, implicitly differentiate to find $v' = (1 - n)y^{-n}y'$, and substitute into the Bernoulli equation
  4. Put the resulting equation into the standard form of a linear differential equation
  5. Find the solution to the linear differential equation, then back-substitute for $v$ and solve for $y$

Example:

Find the solution to the Bernoulli differential equation:

$(y^4 + x^2y) ~dx - 6x^3 ~dy = 0$

Our first step is to rewrite it in standard form.

$(y^4 + x^2y) ~dx = 6x^3 ~dy$

$y^4 + x^2y = 6x^3 \frac{dy}{dx}$

$\frac{dy}{dx} = \frac{y^4 + x^2y}{6x^3}$

Split the fraction so we can separate terms:

$\frac{dy}{dx} = \frac{y^4}{6x^3} + \frac{x^2y}{6x^3}$

$\frac{dy}{dx} - \frac{x^2y}{6x^3} = \frac{y^4}{6x^3}$

$\frac{dy}{dx} - \frac{x^2}{6x^3} y = \frac{1}{6x^3} y^4$

$y' = - \frac{1}{6x} y = \frac{1}{6x^3} y^4$

$\frac{y'}{y^4} - \frac{1}{6x} \frac{y}{y^4} = \frac{1}{6x^3} \frac{y^4}{y^4}$

$y^{-4} y' - \frac{1}{6x} y^{-3} = \frac{1}{6x^3}$

Our substitution is $v = y^{-3}$, so we'll differentiate to get:

$v' = -3y^{-4} y'$

and then solve for $y^{-4} y'$.

$y^{-4} y' = - \frac{1}{3} v'$

Now we can make substitutions into the Bernoulli equation.

$y^{-4} y' - \frac{1}{6x} y^{-3} = \frac{1}{6x^3}$

$- \frac{1}{3} v' - \frac{1}{6x} v = \frac{1}{6x^3}$

Multiplying through by $-3$ puts the equation into standard form of a linear differential equation.

$v' + \frac{1}{2x} v = - \frac{1}{2x^3}$

To find the solution to the linear equation, we'll find the integrating factor

$I(x) = e^{ \int P(x) ~dx }$

$I(x) = e^{ \int \frac{1}{2} x ~dx }$

$I(x) = e^{ \frac{1}{2} ~ln ~x }$

$I(x) = e^{ ln (x^{1/2}) }$

$I(x) = x^{1}{2}$

And then multiply through the linear equation by $I(x)$.

$v' + \frac{1}{2x} v = - \frac{1}{2x^3}$

$v' x^{1/2} + \frac{1}{2x} x^{1/2} v = -\frac{1}{2x^3} x^{1/2}$

$v' x^{1/2} + \frac{1}{2} x^{-1/2} v = - \frac{1}{2x} x^{5/2}$

Reverse the product rule for derivatives to rewrite the left side:

$\frac{d}{dx} (vx^{1/2}) = - \frac{1}{2} x^{-5/2}$

Then integrate both sides:

$\int \frac{d}{dx} (vx^{1/2}) ~dx = \int - \frac{1}{2} x^{-5/2} ~dx$

$vx^{1/2} = - \frac{1}{2} \left( - \frac{2}{3} \right) x^{-3/2} + C$

$vs^{1/2} = \frac{1}{3} x^{-3/2} + C$

Solve for $v$:

$v = \frac{1}{3} \left( \frac{x^{-3/2}}{x^{1/2}} \right) + \frac{C}{x^{1/2}}$

$v = \frac{1}{3x^2} + \frac{C}{x^{1/2}}$

$v = \frac{1 + 3C x^{3/2}}{3x^2}$

If $3C$ is some arbitrary constant, then we can simply use $C$ instead.

$v = \frac{1 + Cx^{3/2}}{3x^2}$

Use $v = y^{-3}$ to back-substitute for $v$.

$y^{-3} = \frac{1 + Cx^{3/2}}{3x^2}$

Then solve for $y$:

$\frac{1}{y^3} = \frac{1 + Cx^{3/2}}{3x^2}$

$y^3 = \frac{3x^2}{1 + C x^{3/2}}$

$y = 3 \sqrt[3]{ \frac{3x^2}{1 + Cx^{3/2}} }$

Homogenous Equations

We've looked at how to use the substitution $v = y^{1-n}$ to change a Bernoulli equation into a linear equation. We'll also use a change of variable to solve homogenous equations, which are differential equations in the form $y' = F \left( \frac{y}{x} \right)$. The equations may require manipulation in order to get into this form.

Once we have the homogenous equation written in standard form, then our goal will be to use a change of variable to rewrite it as a separable differential equation.

Then we'll find the solution to the separable equation, and back-substitute to find the solution to the homogenous equation. In summary, follow these steps:

  1. If the homogenous equation is not already given in standard form, rewrite it in standard form
  2. Substitute $v = \frac{y}{x}$ and $y' = v + xv'$
  3. Solve the resulting separable equation
  4. Back-substitute for $v = \frac{y}{x}$
  5. If possibe, solve explicitly for $y$ to find the solution to the homogenous differential equation

Example:

Find the solution to the following differential equation:

$xyy' = 2x^2 - 3y^2 = 0$

To put the equation in standard form, divide through by $x^2$.

$\frac{xy}{x^2} y' + \frac{2x^2}{x^2} - 3 \frac{y^2}{x^2} = 0$

$\frac{y}{x} y' + 2 - 3 \left( \frac{y}{x} \right) = 0$

$\frac{y}{x} y' = 3 \left( \frac{y}{x} \right)^2 - 2$

Then solve for $y'$:

$y' = 3 \left( \frac{y}{x} \right)^2 \left( \frac{x}{y} \right) - 2 \left( \frac{x}{y} \right)$

$y' = \frac{3}{\frac{y}{x}} \left( \frac{y}{x} \right)^2 - \frac{z}{\frac{y}{x}}$

$y' = 3 \left( \frac{y}{x} \right) - \frac{z}{\frac{y}{x}}$

Now that the equation is in standard form, we'll substitute $v = \frac{y}{x}$ and $y' = v + xv'$

$v + xv' = 3v - \frac{2}{v}$

$v^2 + xvv' = 3v^2 - 2$

$xvv' = 2v^2 - 2$

Now, separate variables:

$xv \frac{dv}{dx} = 2v^2 - 2$

$xv ~dv = (2v^2 - 2) ~dx$

$\frac{v}{2v^2 - 2} ~dv = \frac{1}{x} ~dx$

Then, integrate both sides, using the substitution $u = 2v^2 - 2$, $du = 4v ~dv$, and $dv = \frac{1}{4v} ~du$.

$\int \frac{v}{2v^2 - 2} ~dv = \int \frac{1}{x} ~dx$

$\int \frac{v}{u} \left( \frac{1}{4v} \right) ~du = \int \frac{1}{x} ~dx$

$\frac{1}{4} \int \frac{1}{u} ~du = \int \frac{1}{x} ~dx$

$\frac{1}{4} ~ln |u| = ln |x| + C$

Back-substitute with $u = 2v^2 - 2$

$\frac{1}{4} ln |2v^2 - 2| = ln |x| + C$

$ln|2v^2 - 2| = 4 ~ln|x| + 4C$

$e^{ln|2v^2-2} = e^{4 ~ln|x| + 4C}$

$|2v^2 - 2| = e^{4 ~ln|x|} e^{4C}$

$2v^2 - 2 = Ce^{4 ~ln|x|}$

$2v^2 - 2 = Ce^{ln|x|^4}$

$2v^2 - 2 = C|x|^4$

$2v^2 = C|x|^4 + 2$

$v^2 = C|x|^4 + 1$

$v^2 = Cx^4 + 1$

Then back-substitute with $v = \frac{y}{x}$

$\left( \frac{y}{x} \right)^2 = Cx^4 + 1$

$\frac{y^2}{x^2} = Cx^4 + 1$

$y^2 = Cx^6 + x^2$

$y = \pm \sqrt{ Cx^6 + x^2 }$

Exact Equations

For an equation to be called an exact differential equation, it must be given in the form:

$M(x,y) + N(x,y) \frac{dy}{dx} = 0$

This kind of equation will have an implicit solution $\psi(x,y) = c$, where $c$ is a constant and $\psi$ has continuous partial derivatives such that:

$d \psi(x,y) = M(x,y) ~dx + N(x,y) ~dy$

Test For an Exact Solution

If we say that the functions $M(x,y)$ and $N(x,y)$ have continuous partial derivatives, then the differential equation $M(x,y) + N(x,y) \frac{dy}{dx} = 0$ is exact only if $M_y = N_x$, or:

$\frac{\partial M(x,y)}{\partial y} = \frac{\partial N(x,y)}{\partial x}$

i.e., if the partial derivative of $M$ with respect to $y$ is equal to the partial derivative of $N$ with respect to $x$, then the differential equation is exact.

Solution to an Exact Equation

The implicit solution to an exact differential equation is given by $\psi(x,y) = c$, where $c$ is a constant. The partial derivatives of $\psi$ with respect to $x$ and $y$ are:

$\frac{\partial \psi}{\partial x} = \psi_x = M(x,y) = M$

$\frac{\partial \psi}{\partial y} = \psi_y = N(x,y) = N$

The left sides of the equations are partial derivatives of $\psi$, but to get back to $\psi$ itself, we need to take integrals.

$\int \psi_x ~dx = \int M(x,y) ~dx$

$\int \psi_y ~dy = \int N(x,y) ~dy$

Taking the integral with respect to $x$ of the partial derivative with respect to $x$ cancels both operations.

$\psi = \int M(x,y) ~dx + h(y)$

$\psi = \int N(x,y) ~dy + h(x)$

We're integrating a multivariable function in terms of $x$ and $y$ with respect to $x$ only. This means that instead of adding $c$ to account for the constant of integration after we integrate, we have to add $h(x)$ or $h(y)$. Then it's a matter of solving for $h(y)$ or $h(x)$, by differentiating $\psi$ with respect to $y$ or $x$. That differentiation process will give us either $\psi_x$ with $h'(x)$, or $\psi_y$ with $h'(y)$. We know that $M(x,y) = \psi_x$ or $N(x,y) = \psi_y$, so we'll make that substitution and then simplify the equation to solve for $h(y)$ or $h(x)$.

Finally, we'll plug $h(y)$ or $h(x)$ back into the equation for $\psi$, set the equation equal to $c$, and this will be the implicit solution to the exact differential equation. In summary, to find the solution of an exact differential equation:

  1. Verify that $M_y = N_x$ to confirm the differential equation is exact
  2. Use $\psi = \int M(x,y) ~dx + h(y)$ or $\psi = N(x,y) ~dy + h(x)$ to find $\psi(x,y)$, including a value for $h(y)$ or $h(x)$
  3. Find $\frac{\partial \psi}{\partial y} = \frac{\partial}{\partial y} \left( \int M(x,y) ~dx \right) + h'(y) = N(x,y)$ or find $\frac{\partial \psi}{\partial x} = \frac{\partial}{\partial x} \left( \int N(x,y) ~dy \right) + h'(x) = M(x,y)$
  4. Integrate both sides of the equation to solve for $h(y)$ or $h(x)$
  5. Plug $h(y)$ or $h(x)$ back into $\psi$ (in step 2)
  6. Set $\psi(x,y) = c$ to get the implicit solution

Example:

If the following differential equation is exact, find its solution.

$x^3 y^4 + (x^4 y^3 + 2y) \frac{dy}{dx} = 0$

First, we'll test to see whether or not the differential equation is exact. Matching the given equation to the standard form of an exact differential equation, we find that:

$M(x,y) = x^3 y^4$

$N(x,y) = x^4 y^3 + 2y$

We'll test to see whether $M_y = N_x$.

$\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}$

$\frac{\partial}{\partial y}(x^3 y^4) = \frac{\partial}{\partial x} (x^4 y^3 + 2y)$

$4x^3 y^3 = 4x^3 y^3$

Since $M_y = N_x$, we know that the differential equation is exact, so now we proceed to find the solution $\psi(x,y) = c$.

$\psi = \int M(x,y) ~dx$

$\psi = \int x^3 y^4 ~dx$

$\psi = \frac{1}{4} x^4 y^4 + h(y)$

Now we need to find $h(y)$, which we'll do by differentiating both sides with respect to $y$.

$\psi_y = x^4 y^3 + h'(y)$

Because we know that $\psi_y = N(x,y)$, we'll make that substitution, and then solve for $h'(y)$.

$x^4 y^3 + 2y = x^4 y^3 + h'(y)$

$2y = h'(y)$

To find $h(y)$, we'll integrate both sides of this equation with respect to $y$.

$\int 2y ~dy = \int h'(y) ~dy$

$y^2 + k_1 = h(y) + k_2$

$h(y) = y^2 + k_1 - k_2$

$h(y) = y^2 + k$

Plugging this value for $h(y)$ into the equation for $\psi$ in step 2 gives:

$\psi = \frac{1}{4} x^4 y^4 + h(y)$

$\psi = \frac{1}{4} x^4 y^4 + y^2 + k$

Finally, setting $\psi = c$ to find the solution to the exact differential equation

$c = \frac{1}{4} x^4 y^4 + y^2 + k$

$c - k = \frac{1}{4} x^4 y^4 + y^2$

$c = \frac{1}{4} x^4 y^4 + y^2$

Second Order Equations

Second order equations can take many forms, we will start by looking at second order linear equations that have constant coefficients. Second order linear equations can be homogenous or nonhomogenous. The homogenous variety looks like:

$y'' + Ay' + By = 0$

This is a linear equation because we only see $y$ and its derivatives, and not functions such as $cos ~y$, $e^y$, etc.

Finding the Solution

To find a solution that satisfies a second order linear homogenous equation with constant coefficients, we always start with the associated characteristic equation.

$r^2 = Ar + B = 0$

i.e., we substitute $y'' = r^2, y' = r$, and $y = 1$

This is a quadratic equation, so it will have one of three types of roots - distinct real roots, equal real roots, or complex conjugate roots. The general solution is determined by the type of roots $r_1$ and $r_2$ that we find from the characteristic equation.

The general solution will contain the constants $c_1$ and $c_2$, but we can find the values of $c_1$ and $c_2$ if we are given the initial conditions.

$y'' + 5y' + 6 = 0$

$(r+2)(r+3) = 0$

Reduction of Order

While the values for $r$ are the roots of the characteristic equation, they aren't solutions to the differential equation. Instead, we need to identify the solutions to the differential equation from the equation of the general solution. The solutions to the differential equation will be the functions we see multiplied by $c_1$ and $c_2$.

For instance, given a general solution with distinct real roots:

$y(x) = c_1 e^{r_1x} + c_2 e^{r_2x}$

the solutions to the differential equation are:

$y_1 = e^{r_1x}$

$y_2 = e^{r_2x}$

Alternatively, given a general solution with complex roots:

$y(x) = e^{\alpha x}( c_1 ~cos(\beta x) + c_2 e^{\alpha x} sin(\beta x) )$

$y(x) = c_1 e^{\alpha x} = c_1 e^{\alpha x} cos(\beta x) + c_2 e^{\alpha x} sin(\beta x)$

the solutions to the differential equation are:

$y_1 = e^{\alpha x} cos(\beta x)$

$y_2 = e^{\alpha x} sin(\beta x)$

Notice how all three forms of the general solution can be rewritten in the same form,

$y(x) = c_1 y_1 + c_2 y_2$

Reduction of Order with Constant Coefficients

If we know $y_1$, we can use it to find $y_2$, and vice versa. This is called reduction of order, because it uses a substitution to reduce the order of the homogenous equation to a first order equation.

Example:

Use reduction of order to find the general solution to the differential equation, given $y_1 = e^{x/3}$:

$6y'' + y' - y = 0$

We begin with the assumption that $y_2 = vy_1$, and then find the first and second derivatives of $y_2$, since we are dealing with a second order equation.

$y_2 = ve^{x/3}$

$y'_2 = v' e^{x/3}$

$y''_2 = v'' e^{x/3} + \frac{2}{3}v'e^{x/3} + \frac{1}{9}ve^{x/3}$

Plug these values into the homogenous differential equation:

$6v''e^{x/3} + 4v'e^{x/3} + \frac{2}{3}ve^{x/3} + v'e^{x/3} + \frac{1}{3}ve^{x/3} - ve^{x/3} = 0$

$6v''e^{x/3} + 5v'e^{x/3} = 0$

We'll make a substitution $w = v'$, and therefore $w' = v''$ (this is the reduction of order).

$6w'e^{x/3} + 5we^{x/3} = 0$

This changes the second order equation in v into a first order equation in $w$. This is a linear equation that we can easily solve.

$6e^{x/3} \frac{dw}{dx} + 5e^{x/3} w = 0$

$\frac{dw}{dx} + \frac{5e^{x/3}}{63^{x/3}} w = 0$

$\frac{dw}{dx} + \frac{5}{6} w = 0$

With $P(x) = \frac{5}{6}$, the integrating factor is:

$I(x) = e^{ \int P(x) ~dx }$

$I(x) = e^{ \int \frac{5}{6} x }$

$I(x) = e^{ \frac{5}{6} x }$

Multiplying the first order differential equation in $w$ by the integrating factor, and then simplifying, gives:

$\frac{d}{dx}(we^{ \frac{5}{6} x }) = 0$

Integrate both sides:

$\int \frac{d}{dx}(we^{ \frac{5}{6} x }) ~dx = 0$

$we^{\frac{5}{6} x} = C$

$w = Ce^{ \frac{5}{6} x }$

Because $w = v'$, we'll integrate $w$ to find $v$.

$v = \int w = \int Ce^{ - \frac{5}{6} x } = - \frac{6}{5} Ce^{ - \frac{5}{6} x } + k$

We can choose any constants, so we'll choose $C = - \frac{5}{6}$ and $k=0$ for simplicity. Then $v$ is $v = e^\frac{5}{6}x$, and the second solution for the differential equation is:

$y_2 = ve^{ \frac{1}{3} x }$

$y_2 = e^{ - \frac{5}{6} x } e^{ \frac{1}{3} x }$

$y_2 = e^{ - \frac{x}{2} }$

Because the solutions are $y_1 = e^{x/3}$ and $y_2 = e^{- x/2}$, the general solution to the differential equation is:

$y(x) = c_1 e^{x/3} + c_2 e^{-x/2}$

Undetermined Coefficients For Nonhomogenous Equations

Linear second order homogenous equations with constant coefficients would take the form $y'' = Ay' + By = 0$. Nonhomogenous equations swap out the zero-value on the right side for a non-zero value.

$y'' + Ay' + By = g(x)$

To solve these, we'll first use the characteristic equation to find the general solution to the associated homogenous equation, and then deal with the $g(x)$ part in another step. There are multiple methods we can use, the first of which is the method of undetermined coefficients.

Undetermined Coefficients

The general solution $y(x)$ to a second order nonhomogenous equation will always be the sum of the complementary solution $y_c(x)$ and the particular solution $y_p(x)$.

$y(x) = y_c(x) + y_p(x)$

The complementary solution we find from solving the charateristic equation. Once we find the complementary solution, we use $g(x)$ to make a guess about a particular solution.

  • For a polynomial function like $x^2 + 1$, guess $Ax^2 + Bx + C$
  • For an exponential function like $e^{3x}$, guess $Ae^{3x}$
  • For sine and cosine functions like $3 ~sin(4x)$, guess $A ~sin(4x) + B ~cos(4x)$

If the right side of the differential equation is the sum or product of different types of functions, we need to multiply or add our guesses together. For any overlapping terms, we'll need to multiply that section of the guess by $x$ (or $x^2$, $x^3$, etc.) in order to eliminate the overlap. For example, if the complementary solution includes the term $e^{3x}$, and our guess for the particular solution includes $Ae^{3x}$, then we need to multiply $Ae^{3x}$ by $x$, changing the guess to $Axe^{3x}$. For exponential terms, an overlap only exists if the exponents match exactly; for example, $e^{3x}$ and $Ae^{3x}$ are overlapping, but $e^{3x}$ and $Ae^{5x}$ are not.

Once we've eliminated any overlap and finalized the guess for our particular solution, we'll find the first and second derivatives of our guess, then plug those derivatives into the original differential equation for $y'(x)$ and $y''(x)$. Then we'll be able to combine like terms and equate coefficients on both sides to solve for the constants $A$, $B$, and $C$. We should end up with a particular solution that we can combine with the complementary solution in order to get the general solution to the second order nonhomogenous equation.

Summary of Steps:

  1. Substitute $g(x) = 0$ to rewrite the nonhomogenous equation as a homogenous equation
  2. Use the associated characteristic equation to solve the homogenous equation, generating the complementary solution $y_c(x)$
  3. Make a guess for a particular solution
  4. Fix any overlap between the guess for the particular solution and the complementary solution from step 2
  5. Find the first and second derivatives of the non-overlapping guess for the particular solution
  6. Substitute the derivatives into the original differential equation
  7. Equate coefficients to find the values of any constants
  8. Substitute the constant values into the guess to generate a particular solution $y_p(x)$
  9. Sum the complementary and particular solutions to get the general solution

$y(x) = y_c(x) + y_p(x)$

Example:

Find the general solution to the differential equation:

$y'' + 2y' = 4x - 6e^{-2x}$

In step 1, we'll replace $g(x) = 4x - 6e^{-2x}$ with $g(x) = 0$ to change the nonhomogenous equation into a homogenous equation.

$y'' + 2y' = 0$

Then we'll solve the associated characteristic equation.

$r^2 + 2r = 0$

$r(r+2) = 0$

$r = 0, -2$

Step 2: these are distinct real roots, so the complementary solution will be:

$y_c(x) = c_1e^{r_1x} + c_2e^{r_2x}$

$y_c(x) = c_1e^{(0)x} + c_2e^{-2x}$

$y_c(x) = c_1(1) + c_2e^{-2x}$

$y_c(x) = c_1 + c_2e^{-2x}$

Now we can switch over to working on a particular solution, $g(x) = 4x - 6e^{-2x}$. We have a polynomial function, $4x$, and an exponential function, $-6e^{-2x}$. We'll use $Ax+B$ as our guess for the polynomial function, and $Ce^{-2x}$ as our guess for the exponential function (step 3).

$y_p(x) = Ax + B + Ce^{-2x}$

We see that $c_2e^{-2x}$ from the complementary solution and $Ce^{-2x}$ from the particular solution are overlapping. To fix this, we'll multiply $Ce^{-2x}$ from the particular solution by $x$ (step 4), such that our guess becomes:

$y_p(x) = Ax + B + Cxe^{-2}$

Step 5: taking the first and second derivatives, we get:

$y'_p(x) = 2Ax + B + Ce^{-2x} - 2Cxe^{-2x}$

$y''_p(x) = 2A - 4Ce^{-2x} + 4Cxe^{=2x}$

Step 7: Plugging the first two derivatives into the original differential equation, we get:

$2A + 2B = 0$

$2(1) + 2B = 0$

$2B = -2$

$B = -1$


$4A = 4$

$A = 1$


$-2C = -6$

$C = 3$

Step 8: We'll plug the results into our guess for the particular solution to get:

$y_p(x) = (1)x^2 + (-1)x + (3)xe^{-2x}$

$y_p(x) = x^2 - x + 3xe^{-2x}$

Step 9: Putting this particular solution together with the complementary solution gives us the general solution to the differential equation.

$y(x) = y_c(x) + y_p(x)$

$y(x) = c_1 + c_2e^{-2x} + x^2 - x + 3xe^{-2x}$

Variation of Parameters for Nonhomogenous Equations

Variation of parameters works for a wider family of $g(x)$ functions than undetermined coefficients, but requires some integration, and there is no guarantee we'll get functions that are easy to integrate.

We'll start by substituting $g(x) = 0$ to change the nonhomogenous equation into a homogenous equation. Then we'll use the associated characteristic equation to find the complementary solution.

Complementary SolutionFundamental Solution Set
$y(x) = c_1 e^{r_1x} + c_2 e^{r_2x}$$\{y1, y2\} = \{e^{r_1x}, e^{r_2x}\}$

$u_1' y_1 + u_2' y_2 = 0$

$u_1' y_1' + u_2' y_2' = g(x)$

We'll solve the system for $u_1'$ and $u_2'$, then investigate to find $u_1$ and $u_2$. This is the integration mentioned earlier. If we're able to get $u_1$ and $u_2$, then we can say that a particular solution is:

$y_p(x) = u_1 y_1 + u_2 y_2$

$y(x) = y_c(x) + y_p(x)$

If our differential equation has a $y''$ term with a coefficient that's anything other than $1$, then our first step should be to divide through the equation by that value to change the coefficient on $y''$ to $1$.

Use variation of parameters to find the general solution to the differential equation.

$y'' + 4y' + 4y = \frac{e^{-2x}}{x^3}$

The coefficient on $y''$ is already $1$. We'll substitute $g(x) = 0$ to rewrite the nonhomogenous equation as a homogenous equation.

$y'' + 4y' + 4y = 0$

$r^2 + 4r + 4 = 0$

$r = -2$

$y_c(x) = c_1e^{-2x} + c_2 xe^{-2x}$

$\{y_1, y_2\} = \{e^{-2x}, xe^{-2x}\}$

$u_1' e^{-2x} + u_2' xe^{-2x} = 0$

$u_1' e^{-2x} + \left( \frac{1}{x^3} \right) xe^{-2x} = 0$

$u_1' e^{-2x} + \frac{e^-2x}{x^2} = 0$

$u_1' e^{-2x = - \frac{e^{-2x}}{x^2}}$

$u_1' = - \frac{1}{x^2}$

Integrate $u_1'$ to find $u_1$.

$u_1 = \int u_1' = \int - \frac{1}{x^2} ~dx$

$u_1 = \int - x^{-2} ~dx$

$u_1 = - \frac{1}{-1} x^{-1}$

$u = x^{-1}$

$u_1 = \frac{1}{x}$

And then integrate $u_2'$ to find $u_2$.

$u_2 = \int u_2' = \int \frac{1}{x^3} ~dx$

$u_2 = \int x^{-3} ~dx$

$u_2 = \frac{1}{-2} x^{-2}$

$u_2 = - \frac{1}{2}x^{-2}$

$u_2 = - \frac{1}{2x^2}$

Given these values for $u_1$ and $u_2$, the particular solutions is:

$y_p(x) = u_1 y_1 + u_2 y_2$

$y_p(x) = \frac{1}{x} e^{-2x} - \frac{1}{2x^2} xe^{-2x}$

$y_p(x) = \frac{e^{-2x}}{x} - \frac{e^{-2x}}{2x}$

$y_p(x) = \frac{ 2e^{-2x} - e^{-2x} }{2x}$

$y_p(x) = \frac{e^{-2x}}{2x}$

Adding this particular solution to the complementary solution gives us the general solution $y(x)$.

$y(x) = c_1 e^{-2x} + c_2 xe^{-2x} + \frac{e^{-2x}}{2x}$

Fundamental Set of Solutions

We say that a set of solutions to a differential equation is a fundamental set of solutions when the Wronskian of those solutions is non-zero.

The Wronskian $W$ of a solution set is the determinant of the set of solutions and their derivatives.

$ W(y_1, y_2)(t_0) = det \left( \begin{bmatrix} y_1(t_0) & y_2(t_0) \\ y_1'(t_0) & y_2'(t_0) \\ \end{bmatrix} \right) = y_1(t_0) y_2'(t_0) - y_2(t_0) y_1'(t_0) $

If $y_1(t)$ and $y_2(t)$ are solutions to the differential equation, and if the Wronskain is non-zero $W \neq 0$, then we say the solutions $y_1(t)$ and $y_2(t)$ are linearly independent, which means they are not constant multiples of one another. When the Wronskian is non-zero and the solution set therefore linearly dependent, we call ${y_1, y_2}$ a fundamental set of solutions.

If we have a fundamental set of solutions, we conclude that the general solution is:

$y(t) = c_1 y_1 (t) + c_2 y_2 (t)$

We've been assuming so far that the solution set for a second order equation is a fundamental set, because we've been using any solution set we found to build the general solution, without confirming that $W \neq 0$.

The Reliable Set

There is a set of initial conditions we can use that will always give us a fundamental set of solutions for $y'' + p(t)y' + q(t)y = 0$, which is $y(t_0) = 1$ and $y'(t_0) = 0$, and $y(t_0) = 0$ and $y'(t_0) = 1$.

$ W(y_1, y_2)(t_0) = det \left( \begin{bmatrix} y_1t(0) & y_2(t_0) \\ y_1'(t_0) & y_2'(t_0) \\ \end{bmatrix} \right) = det \left( \begin{bmatrix} 1 & 0 \\ 0 & 1 \\ \end{bmatrix} \right) = 1 - 0 = 1 \neq 0 $

Example:

Use the reliable set of initial conditions to find a fundamental set of solutions for the second order differential equation.

$y'' + 5y' + 6y = 0$

Normally, we would find the general solution to this second order equation by solving the associated characteristic equation:

$r^2 + 5r + 6 = 0$

$(r+3)(r+2) = 0$

$r = -3, -2$

And then plugging these views into the formula for the complementary solution with distinct real roots.

$y_c(x) = c_1 e^{-3x} + c_2 e^{-2x}$

The derivative of this complementary solution is:

$y_c'(x) = -3c_1 e^{-3x} - 2c_2 e^{-2x}$

If we now plug in the initial conditions $y(x_0) = 1$ and $y'(x_0) = 0$, and $y(x_0) = 0$ and $y'(x_0) = 1$, we'll be able to generate another fundamental set of solutions. We'll use $x_0 = 0$, since we can choose any value of $x_0$ that we like, and using $x_0 = 0$ will be easiest.

For $y(0) = 1, 1 = c_1 + c_2$

For $y'(0) = 1, 0 = -3c_1 - 2c_2$

Then this system gives $c_1 = -2$ and $c_2 = -3$, so we could say that one solution to our fundamental set of solutions is:

$y_1(x) = -2e^{-3x} + 3e^{-2x}$

The second set of initial conditions gives:

For $y(0) = 0, 0 = c_1 + c_2$

For $y'(0) = 1, 1 = - 3c_1 - 2c_2$

This gives $c_1 = 1$ and $c_2 = 1$, so the other solution in our fundamental set is:

$y_2(x) = e^{-3x} + e^{-2x}$

So one fundamental set of solutions is:

${y_1, y_2} = {e^{-3x}, e^{-2x}}$

but another is:

$y_c(x) = c_1(-2e^{-3x} + 3e^{-2x}) + c_2(-e^{-3x} + e^{-2x})$

Variation of Parameters with the Wronskian

When we looked at this method before, we built this system of linear equations:

$u_1' y_1 + y_2' y_2 = 0$

$u_1' y_1' + u_2' y_2' = g(x)$

If we solve the first equation for $u_1'$:

$u_1' = - \frac{u_2' y_2}{y_1}$

Then we can substitute it into the second equation to solve for $u_2'$.

$\left( - \frac{u_2' y_2}{y_1} \right) y_1' + u_2' y_2' = g(x)$

$u_2' \left( y_2' - \frac{y_2 y_1'}{y_1} \right) = g(x)$

$u_2' \left( \frac{ y_1 y_2' - y_2 y_1' }{y_1} \right) = g(x)$

$u_2' = \frac{ y_1 g(x) }{ y_1 y_2' - y_2 y_1' }$

Plugging this back into the equation for $u_1'$, we get:

$u_1' = - \frac{ y_1 y_2 g(x) }{ y_1(y_1 y_2' - y_2 y_1') }$

$u_1' = - \frac{ y_2 g(x) }{ y_1 y_2' - y_2 y_1' }$

Notice the denominators of $u_1'$ and $u_2'$ are equivalent, they are both $y_1 y_2' - y_2y_1'$. This value is actually the Wronskian of the fundamental set of solutions ${y_1, y_2}$.

$ W(y_1, y_2) = det \left( \begin{bmatrix} y_1 & y_2 \\ y_1' & y_2' \\ \end{bmatrix} \right) = y_1 y_2' - y_2 y_1' $

We know that the Wronskian of a fundamental set of solutions is non-zero, so the denominators of $u_1'$ and $u_2'$ will be defined.

Now to find the values of $u_1$ and $u_2$, we need to integrate $u_1'$ and $u_2'$.

$u_1 = \int u_1' ~dx$

$= \int - \frac{ y_2 g(x) }{ y_1 y_2' - y_2 y_1' } ~dx$

$= \int - \frac{ y_2 g(x) }{ W(y_1, y_2) } ~dx$

$u_2 = \int u_2' ~dx$

$= \int \frac{ y_1 g(x) }{ y_1 y_2' - y_2 y_1' } ~dx$

$= \int \frac{ y_1 g(x) }{ W(y_1, y_2) } ~dx$

Given these values of $u_1$ and $u_2$, using variation of parameters and these Wronskian integrals tells us that the particular solution to the second order nonhomogenous equation will be:

$y_p(x) = u_1 y_1 + u_2 y_2$

$y_p(x) = - y_1 \int \frac{y_2 g(x)}{W(y_1, y_2)} ~dx + y_2 \frac{y_1 g(x)}{W(y_1, y_2)} ~dx$

Then, just as before, to find the general solution to the second order nonhomogenous equation, we add this particular solution to the complementary solution.

$y(x) = y_c(x) + y_p(x)$

Example:

Use variation of parameters to find the general solution to the differential equation:

$y'' + 4y' + 4y = \frac{e^{-2x}}{x^3}$

As before, we solve the characteristic equation associated with the homogenous equation to find real roots and a complementary solution.

$y_c(x) = c_1 e^{-2x} + c_2 xe^{-2x}$

The fundamental set of solutions and their derivatives are:

${y_1, y_2} = {e^{-2x}, xe^{-2x}}$

${y_1', y_2'} = {-2e^{-2x}, e^{-2x} - 2xe^{-2x}}$

The Wronskian of this fundamental set is:

$ W(e^{-2x}, xe^{-2x}) = det \left( \begin{bmatrix} e^{-2x} & xe^{-2x} \\ -2e^{-2x} & e^{-2x} - 2xe^{-2x} \\ \end{bmatrix} \right) $

$= (e^{-2x}) (e^{-2x} - 2xe^{-2x}) - (xe^{-2x}) (-2e^{-2x})$

$= e^{-4x} - 2xe^{-4x} + 2xe^{-4x}$

$= e^{-4x}$

$ y_p(x) = - e^{-2x} \int \frac{ xe^{-2x} \left( \frac{e^{-2x}}{x^3} \right) }{ e^{-4x} } ~dx + xe^{-2x} \int \frac{ e^{-2x} \left( \frac{e^{-2x}}{x^3} \right) }{ e^{-4x} } ~dx $

$y_p(x) = e^{-2x} \int \frac{1}{x^2} ~dx + xe^{-2x} \int \frac{1}{x^3} ~dx$

$y_p(x) = -e^{-2x} \left( - \frac{1}{x} \right) - \frac{1}{2} xe^{-2x} x^{-2}$

$y_p(x) = \frac{2e^{-2x}}{2x} - \frac{e^{-2x}}{2x}$

$y_p(x) = \frac{e^{-2x}}{2x}$

As before, adding this particular solution to the complementary solution gives us the general solution:

$y(x) = c_1 e^{-2x} + c_2 xe^{-2x} + \frac{e^{-2x}}{2x}$

Initial Value Problems with Nonhomogenous Equations

We've seen how to solve second order nonhomogenous equations using the methods of undetermined coefficients and variation of parameters. Once we've found the general solution as the sum of the complementary and particular solutions, we can extend the problem one step further by solving an initial value problem.

To solve an initial value problem for a second order nonhomogenous differential equation, we'll use undetermined coefficients or variation of parameters to find the general solution:

$y(x) = y_c(x) + y_p(x)$

Then we'll differentiate the general solution to get:

$y'(x) = y_c'(x) + y_p(x)$

Once we have the general solution and its derivative, we'll plug the initial conditions $y(x_0) = y_0$ and $y'(x) = y_1$ into these equations, which will create a system of linear equations that we can solve for $c_1$ and $c_2$. Once we find those, we will substitute them back into the general solution, which will solve the initial value problem.

Example:

Solve the initial value problem for the follwoing second order nonhomogenous equation, given $y(0) = 0$ and $y'(0) = 1$.

$y'' + y' = x - 2e^{-x}$

If we solve the associated characteristic equation, we find roots $r = 0, -1$, which gives us the complementary solution with distinct real roots:

$y_c(x) = c_1 + c_2 e^{-x}$

Our guess for the particular solution will be:

$y_p(x) = Ax^2 + Bx + Cxe^{-x}$

and undetermined coefficients lets us find $A = \frac{1}{2}$, $B = -1$, and $C = 2$, giving:

$y_p(x) = \frac{1}{2}x^2 - x + 2xe^{-x}$

for the particular solution, and therefore a general solution of:

$y(x) = c_1 + c_2 e^{-x} + x - 1 + 2e^{-x} - 2xe{-x}$

Differentiating the general solution gives:

$y'(x) = -c_2 e^{-x} + x - 1 + 2e^{-x} - 2xe^{-x}$

Substitute the initial conditions $y(0) = 0$ and $y'(0) = 1$ into the general solution and its derivative.

$0 = c_1 + c_2 e^{-(0)} + \frac{1}{2}(0)^2 - 0 + 2(0) e^{-(0)}$

$1 = - c_2 e^{-(0)} + 0 - 1 + 2e^{-(0)} - 2(0)e^{-(0)}$

Simplifying these equations gives:

$0 = c_1 + c_2$

$0 = c_2$

Substitute $c_2 = 0$ into $0 = c_1 + c_2$.

$0 = c_1 + 0$

$0 = c_1$

The general solution becomes:

$y(x) = 0 + (0)e^{-x} + \frac{1}{2}x^2 - x + 2xe^{-x}$

$y(x) = \frac{1}{2}x^2 - x + 2xe^{-x}$
