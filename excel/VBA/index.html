Welcome!

This is a resource for understanding the power and practicality of Excel VBA. VBA stands for "Visual Basic Advanced", and is based on the "Visual Basic" language used in MS Office applications; but it is particularly robust and easy to use in (or from) Excel. If you are a frequent user of Excel, who has only dabbled in macros, then this resource was designed for you.
